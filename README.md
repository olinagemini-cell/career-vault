# career-vault

个人职业与知识管理仓库。供 Hermes Agent 读取与分析。

## 目录结构

```
career-vault/
├── career/            ← 跳槽相关
│   ├── resumes/          简历模板 & 投递版本
│   ├── jd_collection/    收集的JD
│   └── interview_prep/   面试准备 & 话术
│
├── legal/             ← 诉讼相关
│   ├── case_materials/   案卷 / 证据清单 / 合同
│   └── court_prep/       庭审策略 / 问题清单
│
├── ai-learning/       ← AI学习资料
│   ├── course_notes/     线下课笔记
│   └── projects/         实践项目
│
├── skills/            ← Hermes Agent技能沉淀（我自动写）
│
└── _archive/          ← 旧文件/不再使用的放这里
```

## 使用方式

- 文件格式：Markdown 优先（.md）
- 文件名：中文或英文，写清楚内容主题
- Hermes 会自动读取并维护索引
