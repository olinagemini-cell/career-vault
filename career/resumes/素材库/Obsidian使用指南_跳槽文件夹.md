# Obsidian使用指南 - 跳槽文件夹

## 快速开始

### 1. 打开关系图
按 `Ctrl+G` 查看你的知识网络图，可以看到：
- 每个核心能力被哪些JD引用
- 哪些JD竞争同一批素材
- 哪些经历还没有被利用

### 2. 使用反向链接面板
打开任意文件，右侧会显示：
- **Backlinks**：哪些文件引用了当前文件
- **Outgoing links**：当前文件引用了哪些文件

## 实用场景

### 场景1：分析新JD
1. 打开JD文件
2. 查看已有链接的能力模块
3. 点击链接跳转到素材库查看详细内容
4. 确定匹配度

### 场景2：准备针对性简历
1. 从[[能力-JD关系索引]]找到目标JD
2. 点击相关能力链接
3. 从素材库复制对应内容
4. 无需来回切换文件

### 场景3：查看哪些JD用某个能力
1. 在素材库中找到能力模块（如"跨境资源争取"）
2. 查看右侧Backlinks面板
3. 看到所有引用此能力的JD列表

### 场景4：发现未使用的经历
1. 按`Ctrl+G`打开关系图
2. 查看孤立节点（没有连接的点）
3. 考虑是否需要在简历中突出这些经历

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+G` | 打开关系图 |
| `Ctrl+Shift+G` | 打开局部关系图 |
| `Ctrl+O` | 快速打开文件 |
| `Ctrl+Shift+O` | 快速搜索符号 |
| `Ctrl+E` | 切换编辑/预览模式 |
| `Ctrl+K` | 插入链接 |

## 下一步建议

### 立即可做
1. 打开Obsidian，按`Ctrl+G`查看关系图
2. 点击任意节点，体验跳转
3. 打开素材库，查看右侧Backlinks面板

### 进阶使用
1. 为更多JD添加回链（复制现有格式）
2. 在素材库中为更多模块添加anchor
3. 使用Tag面板（已启用）按标签筛选

### 推荐插件（可选）
- **Dataview**：自动生成匹配度排名
- **QuickAdd**：一键创建JD分析模板
- **Templater**：高级模板功能
