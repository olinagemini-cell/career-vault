28	Normal	本案不正面请求法院"确认热水设备归业主共有"（法官心理门槛高），而是通过以下路径倒逼法院必须面对产权问题：
29	List Paragraph	质疑成本构成（42元包含什么？）
30	List Paragraph	→ 成本中包含"节能费用"（即利益分成）
31	List Paragraph	→ 利益分成来自能源管理合同
32	List Paragraph	→ 该合同是物业用业主共有设备与第三方签订的
33	List Paragraph	→ 物业有权签吗？
34	List Paragraph	→ 有权的前提：物业拥有设备 或 业主授权
35	List Paragraph	→ 业主从未授权（无业主大会决议）
36	List Paragraph	→ 物业无产权证明（无购买发票、无产权登记、无转移协议）
37	List Paragraph	→ 合同缺乏法律基础 → 利益分成不应摊入业主成本
38	List Paragraph	→ 42元应当降低 → 差额即应退金额
39	Normal	
40	Normal	这条路径的关键：法官要判断"利益分成是否应计入成本"，就必须判断"物业有没有权签这个合同"，就必须触及"设备归谁"。产权问题成为审查成本合理性的必经前提，法官无法回避。

43	Normal	这条线不需要触及产权问题，即使法官完全回避产权，也能独立成立：
44	List Paragraph	物业服务合同约定热水服务是物业义务（第31条+附件3）
45	List Paragraph	→ 业主物业费已覆盖物业履行全部服务义务的人工成本
46	List Paragraph	→ 热水服务（烧水、供水）是物业服务的一部分
47	List Paragraph	→ 人工成本已含在物业费中
48	List Paragraph	→ 住户手册允许物业按"运行成本"另行收费

68	Heading 2	2.1 第一层论证：热水设备归业主共有
69	Normal	热水系统（燃气炉、热水箱、供水管道、水泵、减压阀等）属于建筑区划内的共用设施设备，依法归全体业主共有。

73	Heading 3	法律依据
74	List Paragraph	《民法典》第274条（主力）："建筑区划内的其他公共场所、公用设施和物业服务用房，属于业主共有。"——具体列举条款，直接认定热水设备属于「公用设施」归业主共有，为主力引用条文。
75	List Paragraph	《民法典》第271条（配合）："业主对专有部分以外的共有部分享有共有和共同管理的权利。"——业主基于此享有对热水设备的共有和管理权。
76	List Paragraph	《民法典》第273条："业主对建筑物专有部分以外的共有部分，享有权利，承担义务。"

79	Heading 2	2.2 第二层论证：物业无权商业经营
80	Normal	"两难困境"论证（核心武器）
81	Normal	物业对热水服务的定性只有两种可能，两种都站不住：
82	Normal	
83	Normal	如果是"物业服务"：
84	List Paragraph	服务成本应包含在物业费中，或经业主大会表决确定收费标准
85	List Paragraph	物业服务合同附件3约定物业负责热水系统维护 → 人工成本应在物业费中
86	List Paragraph	物业单方面制定42元/立方米的热水费单价，从未经业主大会表决 → 程序违法
87	Normal	
88	Normal	如果是"商业经营"：
89	List Paragraph	经营许可证在哪里？
90	List Paragraph	与业主签订的热水服务合同在哪里？（服务标准、价格标准、违约责任）
91	List Paragraph	价格备案在哪里？
92	List Paragraph	用的是业主共有设备 → 业主授权决议在哪里？
93	List Paragraph	经营收入归谁？→《民法典》第282条规定归业主共有
94	Normal	
95	Normal	结论：物业享受了经营的全部权利（自主定价、自主改造、签约、分成），却未履行经营的任何义务（无资质、无合同、无标准、无监管、无授权）。
96	Normal	
97	Heading 3	法律依据
98	List Paragraph	《物业管理条例》第55条："利用物业共用部位、共用设施设备进行经营的，应当在征得相关业主、业主大会、物业服务企业的同意后，按照规定办理有关手续。"
99	List Paragraph	《民法典》第282条："建设单位、物业服务企业或者其他管理人等利用业主的共有部分产生的收入，在扣除合理成本之后，属于业主共有。"
100	List Paragraph	《民法典》第278条："下列事项由业主共同决定：……（五）改建、重建建筑物及其附属设施……"
101	Heading 2	2.3 第三层论证：能源管理合同对业主不发生效力

104	Heading 3	四个攻击点
105	List Paragraph	攻击点1：无权代理。物业无权代表业主签订涉及共有设备的15年商业合同；《民法典》第171条：无权代理对被代理人不发生效力；业主从未追认该合同。
106	List Paragraph	攻击点2：未经业主大会表决。15年期限、168万投资、涉及共有设备改造 → 属于重大事项；《民法典》第278条：重大事项由业主共同决定；无业主大会决议 → 程序严重违法。
107	List Paragraph	攻击点3：义务继承条款无效。"物业撤出后义务由业主继承" → 物业无权为业主设定义务；业主从未同意，不受此约束；构成霸王条款（《民法典》第497条）。
108	List Paragraph	攻击点4：收益分配违法。全部"节能效益"由物业和第三方分成，业主零收益；违反《民法典》第282条（共有部分收益归业主共有）。

118	Normal

126	Heading 3	关于"物业亏本经营"的预判和反驳
127	Normal	物业成本表显示连续三年亏损（-196,580/-200,542/-163,159元），物业必然以此辩称"42元还不够成本"。
128	Normal	
129	Normal	三层反驳：
130	List Paragraph	法律层面：亏本不是违法的理由。即使物业确实亏损，也不能倒推出"物业有权未经授权签署合同、将第三方分成摊入业主费用"。亏损是经营结果，不是法律授权。
131	List Paragraph	事实层面：所谓"亏损"建立在一份包含违法成本项目的报表上。成本中包含了不应由业主承担的利益分成和重复人工费——用违法成本证明亏损，再用亏损论证违法成本合理，这是循环论证。
132	List Paragraph	责任层面：物业自己决定签署能源管理合同、自己决定改造方式、自己决定热水费单价42元——所有决策都是物业单方面做的。如果这些决策导致亏损，责任在决策者（物业），不在业主。物业不能把自己决策失误的后果转嫁给业主。
133	List Paragraph	堵住法官的顾虑：即使判决扣除不合理成本项目，物业仍有义务运营热水系统——这是物业服务合同约定的义务（附件3）。如果物业认为运营成本过高，应当通过业主大会协商调整，而不是私下签约、自行定价。解决方案不是继续违法，而是走合法程序。
