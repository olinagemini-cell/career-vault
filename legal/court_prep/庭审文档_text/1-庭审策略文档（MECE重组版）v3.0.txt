第一部分：案件概况
1.1 核心逻辑一句话
物业把业主共有设备当成自己的资产，未经授权与第三方签订15年商业合同私分收益，并将全部成本通过热水费单价转嫁给业主——同时以"服务式公寓"名义收取周边2.25倍的物业费，却声称物业费不包含合同约定的热水服务人工成本。
1.2 诉讼请求
1. 判令被告退还原告2018年9月至2024年6月期间多收的热水费6,083.14元
2. 判令被告承担本案诉讼费用
建议增加的诉讼请求（需律师确认程序可行性）：
3. 判令被告赔偿因擅自拆除热水表导致原告自费安装热水器的损失（以实际发票金额为准）
4. 判令被告恢复原告房屋的热水供应设施至拆除前状态
1.3 关键时间线


1.4 案件定位
本案表面是小额热水费退费纠纷（6,083元），实质目标是通过判决确立以下法律认定：
1. 热水设备属于业主共有的共用设施设备
2. 物业无权未经业主大会授权进行商业经营
3. 能源管理合同中的利益分成不应计入业主热水费成本
第二部分：请求权基础与攻防总图
2.1 请求权基础链
原告的请求权基础（三选一，互为备选）：

路径A：合同违约（物业服务合同）
  物业服务合同附件3约定热水服务义务 + 第6条包干制
  → 物业费已是全部合同义务的对价
  → 物业在热水费中重复收取人工成本 = 合同外多收费用
  → 请求返还多收部分
  法条：《民法典》第577条（违约责任）+ 第509条（合同履行）

路径B：不当得利
  物业将不应由业主承担的成本（节能分成+重复人工费）计入热水费
  → 业主多支付了不应支付的费用
  → 物业获得了没有法律根据的利益
  → 请求返还
  法条：《民法典》第985条（不当得利）

路径C：合同约定（双倍返还）
  物业服务合同第24条约定"擅自提高收费标准的，已支付部分双倍返还"
  → 42元中含不合理成本项 = 实质上提高了收费标准
  → 适用双倍返还
  ⚠️ 此路径文义射程存疑，不作为主攻，仅作补充主张

第三部分：攻击线一——节能费用 / 产权倒逼（主攻）
3.1 逻辑链（7步）
Step 1：42元单价中包含"节能费用"
• 推理：物业成本表中有"节能费用"一栏，该费用被计入热水费成本，由业主承担
• 证据：Excel成本分摊表（物业自己提供） ✅
• 强度：强——物业自己的账目，无法否认
• 漏洞：无
Step 2：节能费用 = 能源管理合同的利益分成
• 推理：Excel"水费/节能费"金额 = PDF结算单中乙方实得金额（全部17个结算周期100%吻合，差额为零）
• 证据：
  • 2022-2024年Q1三年数据：49,471.20 / 42,246.70 / 43,348.01（完全吻合）
  • 2020-2021年8个季度凭证逐季核验（全部吻合）
  • 2022年合计168,048元、2023年合计187,031元（均与年度成本表吻合）
• 强度：极强——17季度铁证如山
• 漏洞：无（"Q1巧合一致"抗辩已被全周期核验彻底封堵）
• 补充发现：2020-2021年与2022-2024年存在记账口径根本矛盾——前期记为物业收入，后期记为支出，同一科目名称性质完全相反，进一步打击成本表可信度
Step 3-4：分成来自能源管理合同，物业以甲方身份签订
• 证据：PDF结算单抬头签章 + 能源管理合同原文 ✅
• 强度：强
• 漏洞：无
合同基本信息：

Step 5：物业签约需具备产权或业主授权 ⚠️ 法官最可能下车点
• 推理：处分/经营共有财产需要业主授权（《民法典》第278条、《物业管理条例》第55条）
• 强度：中强
• 核心漏洞：前提是"热水设备属于业主共有财产"，存在循环论证风险
• 独立证据补强（5层）：
  1. 物业服务合同第31条+附件3（物业自己签字认定"共用设施设备"）
  2. 《住宅说明书》第三章（开发商2004年建设配套设施，原文载明热水设备参数）
  3. 住户手册第四章主动援引说明书（物业自认：热水系统属建筑配套设施）
  4. 预售合同（2003年）附件三"24小时供热水"，第24条物业仅为受托前期管理方
  5. 物业无产权证明（无购买发票、无产权登记、无转移协议）
Step 6：物业既无产权又无业主授权
• 推理：物业无法出示产权证明 + 无业主大会决议
• 强度：中
• 已封堵的漏洞：管理规约全文已审阅（2017年业主大会通过），全文23条无任何授权物业热水经营的条款；第11条反而规定利用共有部分经营须经业主大会同意
Step 7：利益分成不应计入业主成本 → 退费
• 推理：合同缺乏法律基础 → 分成不应由业主承担 → 从成本中扣除 → 单价应降低
• 关键限制：单独扣除成本表上的节能费后（1,112,123 - 168,048 = 944,075），仍超过热水费收入（915,543），账面亏损2.9万 → 必须配合攻击线二
• ⚠️ 但上述"亏损2.9万"本身不成立：物业的成本表只记了支出（向能源公司支付乙方分成），没有记入物业自己从节能结算中获得的甲方40%分成收入（年化约14万元，另有每季度返还人工费8,000元/年化3.2万元）。即：物业既把支付给能源公司的钱算作"成本"让业主承担，又把自己从同一项目中获得的收入藏起来不入账。成本表只记支出不记收入，由此得出的"亏损"结论不具有证明力。
• 庭审策略：不需要精确计算物业隐性收入的具体金额（数据在对方手中），只需要向法庭指出成本表的结构性缺陷——"请被告出示其从能源管理合同中获得的甲方分成收入明细"，由法院审查。

物业从节能结算中的隐性收入（未入账）

物业40%分成 + 扣回人工费的收入未出现在成本表中——物业只记支出不记收入。
四个攻击点
1. 无权代理：物业无权代表业主签订涉及共有设备的15年商业合同（《民法典》第171条）
2. 未经业主大会表决：15年/168万/涉及共有设备改造 = 重大事项（《民法典》第278条）
3. 义务继承条款无效："物业撤出后义务由业主继承"= 霸王条款（《民法典》第497条）
4. 收益分配违法+双重获利：全部"节能效益"由物业和第三方分成，业主零收益（《民法典》第282条）。更恶劣的是：物业支付给第三方的乙方分成，又作为"节能费用"摊入了业主的热水成本表中，由业主买单。也就是说——节能收益100%进了物业的口袋：物业自己拿甲方40%分成（不入账），乙方60%分成由业主通过热水费支付（计入成本表）。物业一分钱没出，却独占全部节能收益。
5. 节能计算模型不可验证：所谓"节能量"的计算基线（假设不改造应花多少）由甲乙双方在合同中自行约定，每季度结算金额也由甲乙双方自行确认——无第三方审计、无政府监管、无业主参与。分成金额的大小完全取决于两个受益方自己搭建的模型和自己签字确认的数字，业主无从验证。
3.3 三方产权悖论（用物业自己的合同打脸）

注：能源管理合同第9.1条和9.3条约定期满设备归甲方（物业），而非业主。设备从业主的（原始）变成物业的（改造后期满），业主始终未被认定为所有权人——这反而印证整个改造方案从未经过业主大会授权。
庭审话术：
"审判长，被告声称热水设备归其所有，但被告自己签署的两份合同恰恰否定了这一点。物业服务合同第31条认定热水系统是共用设施设备——按照《民法典》第274条，这归业主共有。能源管理合同约定改造后的新设备在合同期内归乙方（能源公司）所有。那么请问被告：原始设备是业主的，新设备是能源公司的——被告拥有的是什么？被告凭什么拆除原告门外的热水表？"
3.4 法官下车点与应对
下车点A：Step 5——"是否需要触及产权"（概率60%，风险🔴高）
法官可能说："本案是物业服务合同纠纷，我审查收费是否合理就行，不需要对设备产权作出认定。"
应对：
• 产权归属是审查成本合理性的必经前提，不是独立的诉讼请求
• 给法官低门槛台阶：
"审判长，原告无意请求本院做出超越个案的产权确权判决。原告请求的是：在审查42元/m³热水费单价的合理性时，对成本表中'节能费用'一项是否应由业主承担作出认定。而这一认定的前提，仅需要援引被告自己签署的物业服务合同第31条——该合同已经写明热水系统是共用设施设备。这不是法院在做新的认定，而是适用现有合同条款。"
下车点B：Step 7——"扣不扣节能费，42元还是亏"（风险🟡中）
应对：
1. 指出成本表只记了支出（乙方分成），没有记入物业自己从能源合同中获得的甲方40%分成收入（年化约14万）和返还人工费（年化3.2万）——成本表只记支出不记收入，"亏损"结论不成立
2. 请求被告出示其从能源管理合同中获得的全部收入明细
3. 必须同时打攻击线二（人工费），两项扣除后盈利31.6万
4. 强调违法性独立于盈亏——"亏损不是违法的理由"
下车点C：Step 6——"管理权隐含经营决策权"（风险🟡中）
应对：
1. "三难困境"论证（详见第五部分5.2）
2. 168万投资+15年+利益分成的规模远超"日常管理"
3.5 对方反驳与应对
反驳1："合同第31条是维护义务，不是产权认定"（杀伤力🔴高）
应对：
• 反问对方出示产权证明——物业自己也没有产权登记
• 双方都无产权登记时，物业自己签字确认的"共用设施设备"就是最有力的间接证据
• 结合住宅说明书（开发商2004年配建）和《民法典》第274条
反驳2："节能改造有利于业主，属于善意管理"（杀伤力🟡中）
应对：
• "好意"不能替代"授权"
• 业主零收益——收益被物业和第三方分成
• 节能费占热水费收入比例逐年上升（18.4%→20.7%→22.1%），业主负担不降反升
反驳3："即使扣除节能费，42元仍合理（物业亏损）"（杀伤力🔴高）
应对：
• 首先打掉"亏损"前提：物业成本表只记了向能源公司支付的乙方分成（支出），没有记入物业自己获得的甲方40%分成收入（年化约14万元）和返还人工费（年化3.2万元）。只记支出不记收入，由此得出的"亏损"不具有证明力。请被告出示其从能源合同中获得的全部收入明细。
• 再配合攻击线二。2022年数据：
  • 总支出1,112,123 - 节能费168,048 - 人工费344,663 = 合理成本599,412元
  • 应收热水费915,543 - 合理成本599,412 = 盈利316,131元
反驳4："历年单价表证明42元是延续历史定价"（杀伤力🟡中）
对方数据（物业自己提供的历年单价表，2005-2025年）：
• 2005年25元 → 2007年28元 → 2009年30元 → 2012年35元 → 2013年39元 → 2022年42元
• 物业备注称"能源成本涨幅巨大"为涨价理由
应对：
• 历史定价≠合法定价——程序是否合法与价格水平无关
• 2013-2021年维持39元长达8年，2022年涨到42元——请被告出示2022年涨价的协商记录
• 物业自己的表格备注写了"2015年引入节能系统，能耗费成本明显降低"——既然能耗成本降低了，为什么水费不降？节能效益没有传导给业主
• 2015年前无节能系统时39元，2015年后有节能系统"成本明显降低"仍然39元甚至涨到42元——对业主来说节能系统的"节能"体现在哪里？
反驳5："原告不是适格主体挑战产权和合同效力"（杀伤力🟡中高）
应对：
• 主论点：原告不请求确认合同无效，只请求认定"节能费不应计入业主成本"——这是审查42元单价合理性的必经步骤，不是独立的产权确权或合同效力诉讼
• 产权是审理过程中的必经事实认定，不是独立的诉讼请求
• 《民法典》第274条是强制性规定，任何权利被侵害的业主都可以起诉
• 备选论点（退路）：即使法院不认定能源合同无效，节能项目15年总收益规模约540万元（甲方分成~210万+乙方分成~270万+政府补贴），该收益本应惠及全体业主、拉低热水费。但实际上业主未从节能中获得任何费用减免，反而承担了全部乙方分成支出。退一步讲，至少节能收益中归物业的甲方分成部分（年化约14万元）应当用于冲抵热水运营成本，从而降低热水费单价。
  • ⚠️ 策略提示：此备选论点隐含"承认能源合同有效"的前提，仅在主论点（节能费应删除）被法官否定时使用，不主动提出
第四部分：攻击线二——人工费不应计入热水运行成本
4.1 逻辑链（5步）
Step 1：物业服务合同约定热水是物业服务义务
• 证据：附件3原文"供水系统包含热水系统的日常运行保养和维修"+ "保证热水温度不低于40度"
• 强度：强
• 关键："保证热水温度不低于40度"是结果义务——要达成此结果，必须烧水、值守、调温、供水，已超出单纯"维护设备"范畴
Step 2：热水费中"运行成本"不应包含人工费 ⚠️ 核心命门
主论证路径（首选）：热水费的"运行成本"不应包含人工费
• 住户手册第四章规定"热水费单价由物业根据运行成本制定"
• "运行成本"的自然含义 = 水费、电费、燃气费等直接物料消耗
• 人工费是物业自身的履约成本，不是热水的"运行成本"
• 住户手册由物业单方制定，按《民法典》第498条格式条款规则，"运行成本"应作不利于制定方的解释
• 法规支撑：《物业服务收费管理办法》第11条规定物业服务成本第一项为"管理服务人员的工资"——人工费本就属于物业费的法定构成
补充论证路径：包干制下物业费覆盖全部义务人工
• 合同第6条采用包干制 → 物业费是物业履行全部合同义务的对价
• 附件3将热水系统纳入物业服务范围 → 热水服务人工成本天然在包干制覆盖内
• 三层法规形成包围：

退守论证路径：即使人工费可收，也应从"运行成本"中排除
即使法官不认为"物业费已含热水人工"，仍可论证：物业服务合同约定物业负责热水系统的日常运行，这一义务是物业费的对价之一。物业在履行此义务时产生的人工成本如要另行收取，必须有明确的合同依据或业主大会决议——两者均不存在。
Step 3：热水费成本中确实包含了人工费
• 证据：物业自己的Excel成本分摊表，人工成本数据：
  • 2022年：344,663元（占热水费收入37.6%）
  • 2023年：367,543元（占40.7%）
  • 2024年1-9月：307,499元（占34.7%）
• 强度：铁证，无争议
Step 4：构成不合理收费
• 取决于Step 2是否成立
• 如Step 2的主论证成立（运行成本不含人工）→ 人工费不应出现在热水费成本中

4.3 "热水费独立于物业费"的应对（新增关键章节）
行业背景
根据上海市价格主管部门及《上海市住宅物业服务分等收费管理暂行办法》：
• 物业费覆盖五大项：综合管理、清洁卫生、秩序维护、绿化养护、共用设施日常运行维修
• 热水供应不在上述五项之内，属于独立收费项目
• 热水收费标准不需物价部门审批备案，由物业与业主协商确定
对方必然的攻击
"热水费是独立于物业费的专项服务收费，上海市已明确热水收费不在物业费范围内。物业有权单独收取热水费，人工费计入热水成本完全合理。"
我方多层应对
第一层：热水费独立 ≠ 可以随意定义"运行成本"
"热水费可以独立收取，这我方不否认。但住户手册规定独立收取的范围是'运行成本'。'运行成本'的自然含义是水、电、燃气等直接物料消耗。而被告在成本中不但塞入了物业自身的人力开支——锅炉工薪资占热水费收入的37%至41%；还塞入了与第三方能源公司商业合同的利益分成——以'节能费用'名义列支，占热水费收入的18%至22%。两项合计超过56%。允许热水费独立收取，不等于允许物业把人力开支和商业分成都装进'运行成本'。"
第二层："协商确定" ≠ 物业单方定价
"上海市价格主管部门明确规定，热水收费标准应由物业与业主协商确定。那么请被告出示42元/m³这个单价的协商记录——业主大会讨论记录在哪里？业主投票记录在哪里？公示征求意见的材料在哪里？2022年从39元涨到42元的涨价协商程序在哪里？如果被告无法出示这些记录，则应认定该定价未履行法定的协商程序。"
第三层：4.5元物业费溢价去向的质疑
"即使热水人工费不在物业费范畴内，也请被告向法庭解释：被告收取的物业费4.5元/㎡/月，是周边小区的2.25倍——年溢价约125万元。被告以'服务式公寓'的名义收取远高于住宅标准的物业费，这125万元溢价覆盖了哪些服务？如果连合同约定的热水系统日常运行的人工成本都不覆盖，那这笔溢价的去向是什么？被告不能一边收着远超行业标准的物业费，一边又说物业费不包含合同约定义务的人工成本。"

4.4 对方反驳与应对
反驳1："物业费不包含热水运营成本，热水是独立专项服务"（杀伤力🔴高）
应对：
1. 物业服务合同附件3写了"供水系统包含热水系统"——这就是物业服务的组成部分
2. 如果热水是独立服务，应有独立的服务合同、服务标准、定价程序——请被告出示相关协商记录和业主大会决议
3. 住户手册写的是"运行成本"不是"全部成本"——人工费是物业的管理成本，节能费用是商业合同的利益分成，两者都不属于热水的"运行成本"
4. 4.5元/㎡物业费（周边2.25倍）的溢价去向质疑
反驳2："住户手册授权物业按成本定价，成本包含人工"（杀伤力🟡中）
应对：
1. 住户手册是物业单方制定的文件，不是双方协商的合同
2. "运行成本"应按《民法典》第498条作不利于制定方的解释
3. 即使包含人工，定价权 ≠ 重复收取权——同一笔人工费如已由物业费覆盖（包干制），不能再收
4. 备用论点：住户手册不构成合同（物业可单方修改，不满足《民法典》第469条合同成立要件）
反驳3："人工费金额合理，有薪资证明"（杀伤力🟡中）
应对：
• 真实 ≠ 合理 ≠ 合法——即使每一分钱都是真实支出，问题是"该不该由业主出"
• 成本数据三大矛盾（社保虚列、月度完全相同、涨价悖论）打击可信度
• ✅ 三份锅炉工薪资Excel已分析，新增矛盾四~六（详见4.2节补充）
反驳4："8,000元/季是额外补偿，不是同一笔人工费"（杀伤力🟢低）
应对：
• PDF原文写的就是"扣除人工费"，不是"补偿"
• 这8,000元未从Excel人工成本中扣减——物业同时从两个渠道获取人工费
第五部分：攻击线三——定价程序质疑 + 补充攻击点
5.1 攻击线三：定价程序质疑（零门槛，独立于产权和成本争议）
逻辑链
上海市规定热水收费标准"由物业与业主协商确定"
  + 《民法典》第278条：涉及业主共同利益的重大事项由业主共同决定
  → 原告从未收到过任何定价协商通知或业主大会表决通知
  → 2022年从39元调至42元，原告亦未收到涨价协商或表决通知
  → 请求被告出示：业主大会会议记录、表决记录、协商记录、涨价通知公示等
  → 若被告无法举证 → 定价程序违法成立

价值：这条线不需要法官触及产权、不需要争论成本构成、不需要任何"大动作"——只需要要求被告回答一个事实问题：42元这个数字是怎么来的？有没有经过协商？举证责任在被告一方。
策略提示：⚠️ 原告目前无法完全排除被告曾进行过某种形式的协商或通知（如仅通知部分业主代表），因此本线采取举证倒逼策略——不直接断言"未经协商"，而是要求被告出示协商程序记录，由法院审查其充分性。
庭审话术
"审判长，即使不讨论成本中包含哪些项目，仅从定价程序看：上海市价格主管部门明确规定，小区热水收费标准由物业与业主协商确定。那么我方请求被告回答以下问题：42元/m³这个单价是谁定的？何时定的？与哪些业主协商过？有没有公示征求意见？2022年从39元涨到42元，涨价通知在哪里？业主大会的会议记录和表决记录在哪里？请被告出示相关协商记录和业主大会记录。如果被告无法出示，那么我们可以认定这个价格是被告单方面制定的，未履行协商定价的法定程序。"
5.2 三难困境论证（核心武器）
物业对热水服务的定性只有三种可能，三条路全部走不通：
路径A：如果是"物业服务"：
• 附件3约定"保证热水温度不低于40度" = 结果义务
• 合同第6条包干制 = 物业费是全部义务对价
• → 人工成本应在物业费中，不应在热水费中再收
• → 定价需经业主大会表决或协商程序——请被告出示相关记录
路径B：如果是"增值服务"/"衍生服务"：
• 即使按照上海市物业管理条例，物业可以提供物业合同之外的增值服务
• 但增值服务是基于物业合同本身衍生出来的服务，本质上仍是物业服务的延伸，不是独立商业经营
• 住户手册明确规定热水按"运行成本"核定价格——不是经营利润定价
• → 即使是增值服务，定价也只能按运行成本（水、电、燃气等物料消耗），不能包含商业利润、人工费或能源合同分成
• → 增值服务仍受物业服务法律框架约束，不能脱离物业合同自成一体
路径C：如果是"商业经营"：
• 经营许可证/营业执照中有没有"热水供应"经营范围？请被告出示
• 与业主签订的热水服务合同在哪里？（服务标准、价格标准、违约责任）
• 价格备案在哪里？
• 用的是业主共有设备 → 业主授权决议在哪里？（《物业管理条例》第55条）
• 经营收入归谁？→ 《民法典》第282条规定归业主共有
• 如确为商业经营，需要完整的商业资质和流程：营业执照经营范围、卫生许可、价格备案、独立财务核算、税务处理——请被告逐一出示
结论：物业无论选择哪条路都走不通——
• 选A（物业服务）：包干制已覆盖，人工费不能再收
• 选B（增值服务）：仍按运行成本定价，不能含人工费和商业分成
• 选C（商业经营）：缺乏全部商业资质和授权程序
物业享受了经营的全部权利（自主定价、自主改造、签约、分成），却未履行经营的任何义务（无资质、无合同、无标准、无监管、无授权），也不接受物业服务的约束（运行成本定价、协商程序）。

1. 法律层面：亏本不是违法的理由。即使物业确实亏损，也不能倒推出"物业有权未经授权签署合同、将第三方分成摊入业主费用"

第二，关于人工费。物业服务合同附件3明确约定，被告负责供水系统包含热水系统的日常运行保养和维修，且须保证热水温度不低于40度。住户手册规定热水费按'运行成本'收取——'运行成本'是水、电、燃气等物料消耗，不是人工费。但被告在热水费中收取锅炉工薪资，占热水费收入的37%至41%。

第三，关于节能收益分成。被告的成本明细中包含了来自能源管理合同的利益分成。这份15年期、168万元投资的商业合同，是被告未经业主授权、与第三方签订的。合同产生的成本和分成不应由业主承担。

第四，在本案诉讼期间，被告擅自拆除了原告门外的热水表，导致热水断供。原告诉前热水费全部足额缴纳。

请求法院支持原告的全部诉讼请求。"
11.2 法庭辩论（按顺序展开）
第一轮：定价程序——要求被告举证（零门槛开场）
"审判长，无论成本构成是否合理，我方首先请求被告回答一个程序性问题。上海市价格主管部门明确规定小区热水收费标准由物业与业主协商确定。请被告出示42元/m³定价的协商记录、业主大会会议记录和表决记录。2022年从39元涨到42元，请被告出示涨价的协商记录和业主通知。我方作为业主从未收到过任何定价协商通知。如果被告无法出示这些程序记录，则该定价不符合法定的协商程序。"
第二轮：人工费不应计入热水运行成本
"审判长，我就人工费问题向法庭陈述。住户手册规定热水费按'运行成本'收取。'运行成本'的自然含义是水、电、燃气等直接物料消耗。人工费是物业自身的履约成本——《物业服务收费管理办法》第11条规定物业服务成本第一项就是'管理服务人员的工资'。物业以'服务式公寓'名义收取4.5元/㎡/月的物业费，是周边住宅的2.25倍，年溢价约125万元——如果连合同约定义务的人工成本都不覆盖，这125万元溢价去了哪里？

更严重的是，被告的人工成本数据本身就不可信：2022年每月人工成本精确为28,721.95元，12个月完全相同精确到分——这是预算数字，不是实际发放。个税申报表显示两名锅炉工从未缴纳社保，但成本表中却列入了社保费用。被告向业主收取了从未实际支出的费用。

我方已分析被告提交的三份锅炉工薪资表。结果显示：2022至2024年，王文乾和马春生的社保/公积金栏均为零——连续三年从未缴纳。但德同评估报告却将三人社保和公积金合计每月3,141元列入热水成本，其中至少三分之二是从未发生的虚假支出。此外，2024年仅12个月内先后出现5名所谓'专职锅炉工'，其中一人只干了1个月就消失。年底仅剩2人时，汇总表备注'1人算2人'来凑满3人编制。这不是成本核算，这是成本虚构。"
第三轮：节能费/产权问题
"被告的成本明细中，'节能费用'实际是支付给第三方能源公司的利益分成——全17个季度核验差额为零。这笔分成来自一份15年的能源管理合同。请问被告：签这份合同时，有业主大会的授权吗？用的设备是谁的？物业服务合同第31条和附件3已经写明了——这是共用设施设备。"
第四轮：三难困境+三方产权悖论
"请被告回答一个基本问题：热水服务到底是什么性质？

第一种可能：物业服务。那么合同第6条包干制已经覆盖了全部义务对价，人工费不应再收，定价需经协商程序——请被告出示记录。

第二种可能：增值服务。即使被告主张这是物业合同之外的增值服务，增值服务仍然是基于物业合同衍生出来的服务，住户手册写明按'运行成本'定价——不是经营利润定价。运行成本就是水、电、燃气等物料消耗，不包含人工费和商业合同分成。

第三种可能：商业经营。那么请被告出示：经营许可证中的热水供应经营范围、与业主签订的独立服务合同、价格备案、业主大会授权决议、卫生许可。请问被告，这些材料有吗？

无论被告选哪条路，都无法解释为什么42元单价中包含了人工费和能源合同分成。

再看产权问题：被告如主张热水设备归其所有，请看被告自己签署的两份合同：物业服务合同第31条说是'共用设施设备'——按《民法典》第274条归业主共有；能源管理合同说改造后设备归能源公司所有。原始设备是业主的，新设备是能源公司的——被告拥有的是什么？"
第五轮：回应"亏损"抗辩+拆表事件收尾
"被告声称42元还在亏损。但被告提交的成本表有一个结构性问题：它只记了支出——包括向能源公司支付的乙方分成，却没有记入被告自己从同一能源合同中获得的甲方40%分成收入。请被告出示其从能源管理合同中获得的全部收入明细。只记支出不记收入，由此得出的'亏损'不具有证明力。

即使不考虑隐性收入，扣除节能费和不应计入的人工成本后，2022年的实际盈余是正316,131元，不是亏损196,580元。两个数字相差512,711元。

此外，被告的成本表存在同一年度两份版本数据不一致的问题——请法庭审查。

最后，关于被告在诉讼期间拆除原告热水表一事。原告诉前热水费全额缴清。在对其他业主的同类拆表事件中，派出所民警已明确告知被告'没有依据'——被告不顾制止，继续对多户业主实施同类行为。拆表不是催缴，是惩罚维权的业主。"
11.3 最后陈述
"审判长，本案虽然标的额不大，但涉及的法律问题关系到全小区1,046户业主的切身利益。

我的诉求很简单：物业应当在法律框架内行事。热水费可以收，但价格应当经过与业主的协商程序；成本可以计入，但'运行成本'应当是水电气等物料消耗，不是物业自身的人力开支；设备可以管理，但不能未经授权签订15年商业合同并将成本转嫁给业主。

如果物业认为运营成本高，应当通过业主大会协商调整——解决方案不是绕过程序，而是走合法程序。

恳请法院依法裁判。"
