## 二、收益分成机制（核心问题）

### 分成比例
```
前5年（第1-5年）：  甲方30% | 乙方70%
中5年（第6-10年）： 甲方40% | 乙方60%
后5年（第11-15年）：甲方50% | 乙方50%
```

### 基准线确定机制
- **时间**: 每月10号前
- **方式**: 甲乙双方签字确认
- **内容**: 节能耗能基准线、基准年、基准单价
- **问题**: 
  - ❌ 无业主参与
  - ❌ 无政府监管部门确认
  - ❌ 甲乙双方可以随意调整基准线
  - ❌ "节能效益"计算完全不透明

### 🔴 法律问题
1. **违反《民法典》第282条**
   - 法条: "建设单位、物业服务企业或者其他管理人等利用业主的共有部分产生的收入，在扣除合理成本之后，属于业主共有。"
   - 违法点: 收益完全由物业和第三方分享，业主无份

2. **违反《物业管理条例》第55条**
   - 法条: "利用物业共用部位、共用设施设备进行经营的，应当在征得相关业主、业主大会、物业服务企业的同意后，按照规定办理有关手续。业主所得收益应当主要用于补充专项维修资金，也可以按照业主大会的决定使用。"
   - 违法点: 未征得业主同意，收益未归业主

## 三、权利义务继承条款（陷阱！）

### 合同约定
**原文**: "如果物业因为一些原因撤出了本小区的物业服务工作，这份合同的权利和义务由业主承担，由业主继承。"

### 🔴 法律问题
1. **物业无权代表业主设定义务**
   - 物业只是管理者，不是业主的代理人
   - 未经业主授权，不能为业主设定15年的合同义务

2. **违反《民法典》第278条**
   - 法条: "下列事项由业主共同决定: ...(十一)有关共有和共同管理权利的其他重大事项。"
   - 违法点: 15年合同属于重大事项，必须业主大会表决

3. **违反《民法典》第171条（无权代理）**
   - 法条: "行为人没有代理权、超越代理权或者代理权终止后，仍然实施代理行为，未经被代理人追认的，对被代理人不发生效力。"
   - 违法点: 物业无权代表业主签订此条款，对业主不发生效力

### 💣 攻击要点
- 即使物业撤出，业主也不承担此合同义务
- 这是物业和第三方的"霸王条款"
- 业主从未同意，不受约束

## 四、政府补贴条款

### 合同约定
- 甲方（物业）配合乙方申请政府补贴
- 政府补贴的10%归甲方享受
- 无详细金额和流程规定

### 🔴 法律问题
1. **补贴性质**
   - 节能改造政府补贴是公共资金
   - 目的是鼓励节能，降低社会成本
   - 应当用于降低业主费用，而非物业分成

2. **补贴去向不明**
   - 是否已经申请？
   - 批复金额是多少？
   - 补贴是否已经到账？
   - 补贴如何使用？

3. **物业分成10%违法**
   - 补贴应当用于降低业主费用
   - 物业私自分成，侵害业主权益

### 💣 攻击要点
- 要求物业公开政府补贴申请和批复文件
- 要求物业说明补贴金额和使用情况
- 主张补贴应当用于降低42元/立方米热水费单价

## 五、成本转嫁分析（揭露真相）

### 物业的逻辑
```
第三方投资168万改造设备
    ↓
通过"节能效益"回收投资并盈利
    ↓
物业和第三方按比例分成
    ↓
"节能效益"从哪里来？→ 从业主按42元/立方米单价缴纳的热水费中来！
```

### 实质分析
这不是"节能"，是"转嫁投资成本+商业经营"：

1. **投资方**: 第三方出资168万
2. **回收方式**: 通过向业主收费回收投资
3. **盈利方式**: 物业和第三方分成"节能效益"
4. **买单方**: 业主

### 💰 收益计算（假设）

**假设条件**:
- 小区总户数: 1,046户
- 收费标准: 42元/立方米（按户表用量计费）
- 合同期限: 15年

**年收入**（基于Excel实际数据）:
```
年热水费收入约90万元（2022年实际应收915,543元）
```

**15年总收入**:
```
100,800元/年 × 15年 = 1,512,000元 = 151.2万元
```

**扣除实际运营成本后**:
- 假设实际运营成本（燃气、人工、维护）= 60万元/年
- 15年实际成本 = 60万 × 15 = 900万（这个数字需要根据物业提供的成本明细核实）

**"节能效益"**:
```
如果改造前年成本是80万，改造后年成本是60万
"节能效益" = (80万 - 60万) × 15年 = 300万
```

**分成**:
```
前5年: 300万 × (5/15) × 平均35% = 35万（甲方）+ 65万（乙方）
中5年: 300万 × (5/15) × 平均50% = 50万（甲方）+ 50万（乙方）
后5年: 300万 × (5/15) × 平均50% = 50万（甲方）+ 50万（乙方）

总计: 甲方135万 + 乙方165万 = 300万
```

**关键问题**:
1. 改造前的实际成本是多少？（需要物业提供）
2. 改造后的实际成本是多少？（需要物业提供）
3. "节能效益"是真实的，还是虚构的？
4. 即使有节能效益，为什么不降低业主费用，而是物业和第三方分成？

## 六、五大攻击要点（按优先级）

### 🎯 P0: 合同主体不适格（最强攻击点）

**主张**: 物业无权代表业主签订此合同

**理由**:
1. 热水设备归业主共有（《民法典》第271条）
2. 物业只是管理者，不是所有者
3. 物业无权处分业主的共有财产
4. 物业签订合同属于"无权代理"

**法律依据**:
- 《民法典》第171条（无权代理）
- 《民法典》第271条（共有部分归业主）

**请求**: 确认能源管理合同无效

**攻击力**: ⭐⭐⭐⭐⭐

---

### 🎯 P0: 收益分配违法（最强攻击点）

**主张**: 物业和第三方私分收益，侵害业主权益

**理由**:
1. 《民法典》第282条明确规定：共有部分收益归业主共有
2. 物业和第三方15年分成，业主无份
3. 即使有"节能效益"，也应当用于降低业主费用
4. 物业和第三方的分成是非法所得

**法律依据**:
- 《民法典》第282条
- 《物业管理条例》第55条

**请求**: 
1. 确认收益分配条款无效
2. 物业和第三方返还已分成的收益

**攻击力**: ⭐⭐⭐⭐⭐

---

### 🎯 P0: 程序严重违法（最强攻击点）

**主张**: 未经业主大会表决，程序违法

**理由**:
1. 15年合同属于重大事项
2. 涉及业主共有财产的处分和使用
3. 必须经过业主大会表决
4. 物业单方面签订，程序违法

**法律依据**:
- 《民法典》第278条
- 《物业管理条例》第11条、第12条

**请求**: 确认合同因程序违法而无效

**攻击力**: ⭐⭐⭐⭐⭐

---

### 🎯 P1: 义务转嫁无效（重要攻击点）

**主张**: "物业撤出后义务由业主继承"条款无效

**理由**:
1. 物业无权为业主设定义务
2. 业主从未同意承担此义务
3. 即使物业撤出，业主也不受此约束
4. 这是典型的"霸王条款"

**法律依据**:
- 《民法典》第171条（无权代理）
- 《民法典》第497条（格式条款无效）

**请求**: 确认义务继承条款对业主不发生效力

**攻击力**: ⭐⭐⭐⭐

---

### 🎯 P1: 政府补贴去向不明（重要攻击点）

**主张**: 政府补贴应当用于降低业主费用，而非物业分成

**理由**:
1. 政府补贴是公共资金，目的是降低社会成本
2. 补贴应当惠及业主，降低收费标准
3. 物业私自分成10%，违法
4. 补贴去向不明，需要公开

**法律依据**:
- 政府补贴管理相关规定
- 《民法典》第282条（收益归业主）

**请求**: 
1. 物业公开政府补贴申请和使用情况
2. 补贴应当用于降低42元/立方米热水费单价
3. 物业返还已分成的补贴款

**攻击力**: ⭐⭐⭐⭐

## 九、庭审策略建议

### 开场陈述（3分钟）

```
尊敬的审判长:

本案的核心是一份未经业主同意、侵害业主权益的非法合同。
```

## 十、核心结论

这份能源管理合同是物业和第三方利用业主共有设备进行商业经营、私分收益的典型案例：

1. **合同主体不适格**: 物业无权代表业主签订
2. **收益分配违法**: 物业和第三方私分，业主无份
3. **程序严重违法**: 未经业主大会表决
4. **义务转嫁无效**: 物业无权为业主设定义务
5. **政府补贴去向不明**: 补贴应当惠及业主

**诉讼策略**: 确认合同无效 → 物业和第三方返还收益 → 推翻42元收费标准

**关键逻辑链**:
```
合同无效 
  → 168万投资风险由物业和第三方承担 
    → 不应转嫁给业主 
      → 42元中包含的改造成本分摊无效 
        → 收费标准应当恢复或重新协商
```
