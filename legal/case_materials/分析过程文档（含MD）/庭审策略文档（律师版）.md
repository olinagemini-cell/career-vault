## 第一部分：案件概况

### 1.1 案件定位

本案表面是小额热水费退费纠纷（6,083元），实质目标是通过判决确立以下法律认定：
1. 热水设备属于业主共有的共用设施设备
2. 物业无权未经业主大会授权进行商业经营
3. 能源管理合同中的利益分成不应计入业主热水费成本

### 1.2 一句话核心逻辑

> **物业把业主共有设备当成自己的资产，未经授权与第三方签订15年商业合同私分收益，并将全部成本通过42元/立方米的热水费单价转嫁给业主。**

### 1.3 现有诉讼请求

1. 判令被告退还原告2018年9月至2024年6月期间多收的热水费6,083.14元
2. 判令被告承担本案诉讼费用

### 1.4 建议增加的诉讼请求（需律师确认程序可行性）

3. 判令被告赔偿因擅自拆除热水表导致原告自费安装热水器的损失（以实际发票金额为准）
4. 判令被告恢复原告房屋的热水供应设施至拆除前状态

## 第二部分：核心法律论证

### 战略说明：双线并行攻击 + 补充攻击点

本案采用**两条独立攻击线**，确保无论法官是否触及产权问题，至少有一条线能独立成立。同时增加一个**补充攻击点**，用物业自己的合同打物业自己的脸。

---

#### 攻击线一：节能费用 → 倒逼产权认定（主攻）

本案不正面请求法院"确认热水设备归业主共有"（法官心理门槛高），而是通过以下路径**倒逼法院必须面对产权问题**：

```
质疑成本构成（42元包含什么？）
  → 成本中包含"节能费用"（即利益分成）
    → 利益分成来自能源管理合同
      → 该合同是物业用业主共有设备与第三方签订的
        → 物业有权签吗？
          → 有权的前提：物业拥有设备 或 业主授权
            → 业主从未授权（无业主大会决议）
            → 物业无产权证明（无购买发票、无产权登记、无转移协议）
              → 合同缺乏法律基础
                → 利益分成不应摊入业主成本
                  → 42元应当降低 → 差额即应退金额
```

**这条路径的关键**：法官要判断"利益分成是否应计入成本"，就必须判断"物业有没有权签这个合同"，就必须触及"设备归谁"。产权问题成为审查成本合理性的**必经前提**，法官无法回避。

---

#### 攻击线二：人工费重复收取（独立于产权问题，最稳路径）

这条线**不需要触及产权问题**，即使法官完全回避产权，也能独立成立：

```
物业服务合同约定热水服务是物业义务（第31条+附件3）
  → 业主物业费已覆盖物业履行全部服务义务的人工成本
    → 热水服务（烧水、供水）是物业服务的一部分
      → 人工成本已含在物业费中
        → 住宅说明书/住户手册允许物业按"运营成本"另行收费
          → 但"运营成本"= 水费、电费、燃气费等物料消耗
            → 不应包含人工费
              → 热水费成本中单独列出人工成本（占37-41%）
                → 构成重复收费 → 应扣除
```

**这条线的价值**：
- **零门槛**：不依赖产权认定，不需要法官做任何"大动作"
- **证据现成**：物业服务合同附件3白纸黑字 + 物业自己的成本分摊表
- **数额可观**：人工成本占热水费收入的37-41%，三年合计约102万元
- **逻辑封闭**：即使法官说"42元总体合理"，也无法解释"为什么同一笔人工费要交两次"

**与攻击线一的配合**：两条线独立但互补。攻击线一打产权和合同效力（高收益但法官可能回避），攻击线二打重复收费（稳定输出不依赖法官态度）。法庭辩论中先出攻击线二（稳），再出攻击线一（狠）。

---

#### 补充攻击点：三方产权悖论（用物业自己的合同打物业的脸）

物业如果声称"热水设备归物业所有"，其自己签署的**两份合同自相矛盾**：

| 设备类别 | 物业的文件 | 结论 |
|---------|-----------|------|
| **原始设备**（燃气炉、水箱、管道） | 物业服务合同第31条：认定为"共用设施设备" | → 归业主共有 |
| **新改造设备**（168万投资部分） | 能源管理合同：合同期内（15年）归乙方（能源公司）所有 | → 归能源公司 |
| **物业声称拥有的** | 无任何产权证明 | → 物业什么都不拥有 |

**逻辑闭环**：
```
原始设备 → 物业服务合同说是"共用设施设备" → 归业主共有
新改造设备 → 能源管理合同说归乙方（能源公司）→ 合同期内归能源公司
物业 → 两头都不沾 → 既不拥有原始设备，也不拥有新设备
  → 物业凭什么处置这些设备？
  → 物业凭什么拆除原告的热水表？
```

**庭审使用价值**：极高。这不是我们的推测，而是物业自己签的两份合同导出的必然结论。法官和对方律师都无法反驳——除非物业推翻自己签过的合同。

**庭审话术**：
> "审判长，被告声称热水设备归其所有，但被告自己签署的两份合同恰恰否定了这一点。物业服务合同第31条认定热水系统是共用设施设备——按照《民法典》第274条，这归业主共有。能源管理合同约定改造后的新设备在合同期内归乙方（能源公司）所有。那么请问被告：原始设备是业主的，新设备是能源公司的——被告拥有的是什么？被告凭什么拆除原告门外的热水表？"

### 2.1 第一层论证：热水设备归业主共有

#### 论点
热水系统（燃气炉、热水箱、供水管道、水泵、减压阀等）属于建筑区划内的共用设施设备，依法归全体业主共有。

#### 证据链

| 序号 | 证据 | 内容 | 证明目的 |
|------|------|------|---------|
| 1 | **物业服务合同第31条** | "共用设施设备"包括"供水管道、水箱、水泵" | 物业自己签的合同，承认热水系统是共用设施设备 |
| 2 | **物业服务合同附件3** | "供水系统包含热水系统，热水温度不低于40度" | 明确热水系统属于供水系统的一部分 |
| 3 | **物业服务合同第15条** | "利用小区共用部分获取的收入扣除管理成本后的收益……归全体业主所有" | 节能收益应归业主，物业自己拿40%分成违反本条 |
| 4 | **住宅说明书**（2004年竣工） | 列明"常压燃气炉"、"屋顶热水箱"等设备参数 | 设备由开发商建设，属于建筑配套设施 |
| 5 | **预售合同附件三**（2003年签署，房产交易所备案） | "24小时供热水"——开发商对购房人的交付承诺 | 热水系统是房价的组成部分，非物业自建经营资产 |
| 6 | **管理规约第11条**（2017年业主大会通过） | "利用物业共有部分、共用设施设备从事经营的，应当在征得相关业主、业主大会和物业服务企业的同意后，按规定办理有关手续" | 规约本身承认热水系统是共有部分；经营须经业主大会同意 |
| 7 | **物业无产权证明** | 物业未提供购买发票、产权登记、转移协议 | 物业无法证明设备归其所有 |

#### 法律依据

- **《民法典》第274条**（主力条文）："建筑区划内的其他公共场所、公用设施和物业服务用房，属于业主共有。"
- **《民法典》第271条**（配合条文）："业主对建筑物内的住宅、经营性用房等专有部分享有所有权，对专有部分以外的共有部分享有共有和共同管理的权利。"
- **《民法典》第273条**："业主对建筑物专有部分以外的共有部分，享有权利，承担义务。"

#### 逻辑闭环

```
物业服务合同第31条：供水管道、水箱、水泵 = 共用设施设备
  +
附件3：供水系统包含热水系统
  =
热水系统 = 共用设施设备
  +
《民法典》第274条：公用设施 = 业主共有
  =
结论：热水设备归业主共有，物业只是管理者
```

#### 物业可能的反驳及应对

| 物业说法 | 我方反驳 |
|---------|---------|
| "住宅说明书写了'由物业负责收费'" | "负责收费"是管理职责，不是所有权。物业也负责收物业费，不代表物业拥有整栋楼 |
| "我们一直在管理热水设备" | 管理权≠所有权。物业管理电梯不代表拥有电梯，管理绿化不代表拥有绿地 |
| "设备在我们账上" | 账面记录≠法律产权。请出示购买发票、产权登记或转移协议 |
| "住户手册写了'单价由物业制定'" | 住户手册可作参考，但"运行成本"应按《民法典》第498条作不利于制定方的解释——不应包含商业分成和已由物业费覆盖的人工费 |
| **"热水设备归物业所有"** | **三方产权悖论**：被告自己签的两份合同自相矛盾——物业服务合同第31条认定热水系统是"共用设施设备"（归业主共有）；能源管理合同约定新改造设备在合同期内归乙方（能源公司）所有。原始设备是业主的，新设备是能源公司的——被告什么都不拥有，凭什么处置？ |

### 2.2 第二层论证：物业无权商业经营

#### 论点
物业将热水供应从"物业服务"转变为"商业经营"（自主定价、签订商业合同、与第三方分成），但缺乏经营的全部法律要件。

#### "两难困境"论证（核心武器）

物业对热水服务的定性只有两种可能，**两种都站不住**：

**如果是"物业服务"：**
- 物业服务合同附件3不仅约定物业负责热水系统维护，还明确要求**"保证热水温度不低于40度"**——这是结果义务，意味着整个热水服务（烧水、值守、调温、供水）都已被纳入物业合同义务范围，而非仅仅是"修设备"
- 合同第六条采用**包干制**：物业费是物业履行全部合同义务的对价，盈亏自负 → 履行"保证水温40度"这一义务所需的全部人工成本，天然包含在物业费中
- 因此：人工成本应在物业费中，不应在热水费中再次收取
- 物业单方面制定42元/立方米的热水费单价，从未经业主大会表决 → 程序违法

**如果是"商业经营"：**
- 经营许可证在哪里？
- 与业主签订的热水服务合同在哪里？（服务标准、价格标准、违约责任）
- 价格备案在哪里？
- 用的是业主共有设备 → 业主授权决议在哪里？
- 经营收入归谁？→ 《民法典》第282条规定归业主共有

**结论**：物业享受了经营的全部权利（自主定价、自主改造、签约、分成），却未履行经营的任何义务（无资质、无合同、无标准、无监管、无授权）。

#### 法律依据

- **《物业管理条例》第55条**："利用物业共用部位、共用设施设备进行经营的，应当在征得相关业主、业主大会、物业服务企业的同意后，按照规定办理有关手续。"
- **《民法典》第282条**："建设单位、物业服务企业或者其他管理人等利用业主的共有部分产生的收入，在扣除合理成本之后，属于业主共有。"
- **《民法典》第278条**："下列事项由业主共同决定：……（五）改建、重建建筑物及其附属设施……"

### 2.3 第三层论证：能源管理合同对业主不发生效力

#### 合同基本信息

| 项目 | 内容 |
|------|------|
| 甲方 | 上海世纪商务大厦物业管理有限公司（被告） |
| 乙方 | 上海济通中普能源科技发展有限公司 |
| 投资总额 | 168万元（乙方全额投资） |
| 合同期限 | 15年（约2015-2029年） |
| 分成比例 | 前5年：甲30%/乙70%；后5年：甲40%/乙60%；最后5年：甲50%/乙50% |
| 义务继承 | 物业撤出后，合同义务由业主继承 |
| 政府补贴 | 物业配合申请，补贴的10%归物业 |

#### 四个攻击点

**攻击点1：无权代理**
- 物业无权代表业主签订涉及共有设备的15年商业合同
- 《民法典》第171条：无权代理对被代理人不发生效力
- 业主从未追认该合同

**攻击点2：未经业主大会表决**
- 15年期限、168万投资、涉及共有设备改造 → 属于重大事项
- 《民法典》第278条：重大事项由业主共同决定
- 无业主大会决议 → 程序严重违法

**攻击点3：义务继承条款无效**
- "物业撤出后义务由业主继承"→ 物业无权为业主设定义务
- 业主从未同意，不受此约束
- 构成霸王条款（《民法典》第497条）

**攻击点4：收益分配违法**
- 全部"节能效益"由物业和第三方分成，业主零收益
- 违反《民法典》第282条（共有部分收益归业主共有）

### 2.4 第四层论证：42元成本构成包含不合理项目

#### 论点
即使不讨论产权和合同效力，仅从成本构成看，42元/立方米的热水费单价中包含两类不应由业主承担的费用：

**不合理项目1：利益分成（节能费用）**
- 物业成本表中"节能费用"实为支付给能源公司的利益分成
- 2022年：168,048元（占热水费收入18.4%）
- 2023年：187,031元（占20.7%）
- 2024年：195,228元（占22.1%）
- 该分成来自一份未经业主授权的商业合同，不应由业主承担

**不合理项目2：人工费重复收取**
- 物业服务合同附件3约定：物业负责"供水系统包含热水系统"的日常运行保养和维修
- 这些人工成本应包含在物业费中
- 但物业在热水费成本中又单独列出人工成本（2022年：344,663元，占37.6%）
- 构成重复收费

#### 关于"物业亏本经营"的预判和反驳

物业成本表显示连续三年亏损（-196,580 / -200,542 / -163,159元），物业必然以此辩称"42元还不够成本"。

**三层反驳：**

**法律层面**：亏本不是违法的理由。即使物业确实亏损，也不能倒推出"物业有权未经授权签署合同、将第三方分成摊入业主费用"。亏损是经营结果，不是法律授权。

**事实层面**：所谓"亏损"建立在一份包含违法成本项目的报表上。成本中包含了不应由业主承担的利益分成和重复人工费——用违法成本证明亏损，再用亏损论证违法成本合理，这是**循环论证**。

**责任层面**：物业自己决定签署能源管理合同、自己决定改造方式、自己决定热水费单价42元/立方米——所有决策都是物业单方面做的。如果这些决策导致亏损，责任在决策者（物业），不在业主。物业不能把自己决策失误的后果转嫁给业主。

**堵住法官的顾虑**：即使判决扣除不合理成本项目，物业仍有义务运营热水系统——这是物业服务合同约定的义务（附件3）。如果物业认为运营成本过高，应当通过业主大会协商调整，而不是私下签约、自行定价。**解决方案不是继续违法，而是走合法程序。**

### 3.5 成本表的证明力问题——两份公示版本自相矛盾（五层独立问题）

物业提交的热水成本分摊表，存在以下**五层独立问题**，单独任一层均足以动摇其证明力，五层叠加则足以请求法院不予采信。

#### 第一层：同一年度存在两份数字不同的盖章公示版本

以2023年为例，原告持有两份均加盖上海世纪商务大厦物业管理有限公司公章、署名"制表人：朱艳"的热水成本分摊表，两者数字不一致：

| 对比项 | 版本A（Excel电子版） | 版本B（纸质盖章公示版） | 差额 |
|--------|-------------------|---------------------|------|
| 支出合计 | 1,103,457.73元 | 1,110,650.30元 | +7,192.57元 |
| 利润（亏损） | -200,541.73元 | -207,734.30元 | -7,192.57元 |
| 3月支出 | 219,127.62元 | 225,770.71元 | +6,643.09元 |
| 节能费列 | 187,030.74元（单独一列） | **无此列（整列消失）** | — |
| 银座电费列 | 244,343.88元（单独一列） | **无此列（整列消失）** | — |
| 锅炉电费合计 | 38,976.86元 | **477,544.05元** | +438,567.19元 |

> **数字逻辑验证**：版本A（电费38,976 + 银座电费244,344 + 节能费187,031）= 470,351元；版本B锅炉电费合计 = 477,544元；差额 = 7,193元 ≈ 两版本支出合计之差（7,192.57元）。结论：版本B将节能费和银座电费整列删除，并入"锅炉电费"列，但金额未能完全对齐，留下了7,192元的内部矛盾——这是自相矛盾的直接证据。

#### 第二层：节能费在对外公示版本中被刻意隐藏

版本B是物业向业主张贴公示的纸质版本，版本A是物业向原告提供的电子版本。两者的核心差异在于：
- **公示版（版本B）**：无节能费列，无银座电费列，业主无从知晓这两项合计43万元的争议支出
- **对原告版（版本A）**：节能费187,030元单独列出，银座电费244,344元单独列出

物业向业主公示的版本**刻意删去了节能费一列**——这本身就说明物业知晓节能费无法向业主正当说明，因此在公开文件中将其隐藏，却通过虚增"锅炉电费"来维持支出总数。这一行为构成对业主知情权的主动侵害。

#### 第三层：银座电费三年完全相同——显示非实际计量

| 年份 | 银座电费 | 与前年差额 |
|------|---------|-----------|
| 2022年 | 244,309.54元 | — |
| 2023年 | 244,343.88元 | +34.34元 |
| 2024年 | 244,585.13元 | +241.25元 |

三年最大差额仅275元。电费受气温、用量、电价等多重因素影响，正常年份不可能三年几乎完全相同。合理解释只有一种：银座电费并非来自实际电表计量，而是按固定预算数字填入。原告已要求提供独立电表计费记录，如物业无法出示，则该项支出不具有真实性。

#### 第四层：人工成本月度分布存在异常集中

成本表显示，部分年度Q4人工成本骤然攀升，与正常月份差距悬殊：
- 2021年10-12月人工成本合计142,717元，占全年（389,169元）的36.7%，正常月份月均约27,000元，Q4月均达47,572元（约为正常月1.76倍）
- 2022年12月维修费单月录入24,670.60元，占全年（34,421.25元）的71.7%，其余11个月合计仅9,750元

上述数据表明成本表存在年末集中填报的规律，与真实发生的业务不符，疑似年末为达到目标亏损额而追加录入。

#### 第五层：7年账面持续亏损但物业从未披露、从未申请调价

如果成本表反映的是真实亏损，物业依据《物业服务收费管理办法》及合同约定，有义务向业主大会公示成本并申请调价。但物业在7年间：(1)从未公示成本（业主直至2024年才通过业主群获知）；(2)从未申请调价；(3)从未降低热水服务标准。

更关键的是：物业在账面亏损期间，同步从节能收益分成中额外获得收益（仅2020-2022三年可查金额约为263,662元），该收入从未计入成本表的"应收热水费"一栏。将节能分成收入还原后，物业实际并无亏损。

> **庭上质证话术**：
> "被告提交的热水成本分摊表，同一年度存在两份数字不同、列结构不同的盖章版本；公示版本整列删除了节能费；银座电费三年完全相同显示非实际计量；年末集中录入人工费和维修费；以及7年亏损但从未依规公示或申请调价——五层问题叠加，说明该成本表是物业单方面制作的内部核算工具，不具有证明热水成本客观真实性的证据资格，请法院不予采信。"

### 4.2 物业证据的质证意见

#### 对"住宅说明书"的质证

物业主张："住宅说明书写了'热水表计费收费由物业负责'"

**质证意见**：
> 第一，该说明书是开发商上海荣联房地产有限公司于2004年建设竣工时依法提供给购房人的法定技术文件，由开发商制作，不具备为物业公司创设超出物业服务合同范围收费权的效力。

> 第二，"热水表计量收费由物业负责"中"负责收费"的字面含义是委托物业代为按表计量并执行收费，即管理责任的委托。这绝不等于：①授权物业在物业费之外另立独立收费项目；②授权物业自主制定热水单价；③授权物业将商业分成计入成本向业主转嫁；④授权物业签订15年期商业合同并从中分成。
>
> 第三，说明书第三章恰恰证明了热水设备（常压燃气锅炉、屋顶热水箱、减压阀等）是开发商建设的建筑配套设施，根据《民法典》第274条归业主共有。
>
> 第四，说明书记载"每户设热水表各一个"，证明原本可以按户计量；物业2015年改造后改称"无法按户计量"，与说明书记载矛盾。
>
> 第五，物业自制住户手册第四章主动注明"具体情况见《新建住宅使用说明书》"——物业自己援引说明书，不能在庭上主张说明书不能约束自己。

#### 对"住户手册"的质证

物业主张："住户手册写了'热水费单价由物业根据运行成本制定'"

**住户手册法律定位（方案B：承认参考效力，限制解释范围）**：

住户手册可作为了解物业服务内容的参考文件，但其效力受以下三重限制：

1. **不能对抗上位法和物业服务合同**：物业服务合同是双方签字的正式合同，住户手册是物业单方编制的告知性文件（前言原文："本手册之编制目的在于使各业主/住户了解本公寓的一般管理服务范围及内容"，"管理处保留解释及修改条文及规定的权利"）。当两者冲突时，以物业服务合同为准。

2. **"运行成本"应作不利于制定方的解释**（《民法典》第498条格式条款规则）：住户手册由物业单方制定，对"运行成本"一词存在歧义时，应作有利于业主的解释。"运行成本"的通常文义是水费、电费、燃气费等直接物料消耗，不应包含：
   - 物业与第三方的商业分成（节能费）——这是商业经营成本，不是运行成本
   - 已由物业费覆盖的人工成本——这是物业履行合同义务的自身成本，不是热水运行成本

3. **"定价权"不等于"重复收取权"**：即使承认住户手册赋予物业按运行成本定价的权力，该权力也仅限于在合理范围内确定单价，不授权物业将已由物业费承担的成本再次向业主收取，更不授权将未经业主授权的商业合同分成纳入成本。

**备用论点（如对方紧咬不放）**：住户手册根本不构成合同——合同须经要约与承诺双方合意形成（《民法典》第469条），住户手册是单方告知文件，物业可单方修改（前言原文），不满足合同成立条件。

**质证意见**：
> 我方不否认住户手册记载了"热水费单价由物业根据运行成本制定"。但"运行成本"的合理范围是水、电、燃气等直接消耗，不应包含两类费用：第一，未经业主授权的商业合同分成（节能费）；第二，已由物业费覆盖的锅炉工人工成本。住户手册由物业单方制定，按《民法典》第498条应作有利于业主的解释。更重要的是，定价权不等于重复收取权——即使物业有权按运行成本定价，也不能将同一笔人工费在物业费和热水费中各收一次。

#### 对"成本明细"的质证

物业主张："42元是按成本核算的，合理"

**质证意见**：
> 第一，成本中"节能费用"实为能源管理合同的利益分成（已通过PDF结算单和Excel数据完全吻合验证），该合同未经业主授权，分成不应计入业主成本。第二，人工成本应包含在物业费中（物业服务合同附件3约定物业负责热水系统维护），在热水费中重复收取构成双重收费。第三，即使成本是真实的，物业也无权单方面定价——定价程序违反《民法典》第278条。

#### 对"成本公示"的质证（重要：两份矛盾的成本表）

物业可能主张："物业每年都在公告栏公示成本明细，业主可以随时查看"

**质证意见**：
> 被告声称公示成本表，但原告持有被告**在同一时间张贴的、覆盖同一服务期间的两份成本表，两份表的数据却不一致**。同一个时间段的热水成本，被告自己出了两个不同的版本——请问哪一份是真实的？如果被告连自己的成本数据都前后矛盾，法庭如何信任被告提供的任何成本数据？
>
> 这恰恰证明了三个问题：第一，被告的成本核算不严谨，数据不可靠；第二，所谓的"公示"只是走形式，公示的数据本身就不准确，业主即使看了也无法获得真实信息；第三，被告据以主张"42元合理"的成本数据的真实性和准确性存疑，不应作为定价合理性的证据采信。

**💡 庭审策略提示**：这份证据不要在质证阶段就亮出来——应当**留到法庭辩论阶段**，在物业用成本数据辩称"42元合理"之后再拿出来。先让物业把"成本合理"说透、说死，然后用这两份矛盾的表当场打脸，效果最大化。

#### 对"能源管理合同"的质证

物业主张："合同是为了节能降耗，对业主有利"

**质证意见**：
> 第一，有利与否应由业主自行判断，不是物业单方面决定。第二，如果真的节能，为什么不降低业主费用，反而物业和第三方分成利润？第三，即使有利也必须经过业主大会表决——"好意"不能替代"授权"。第四，合同约定"物业撤出后义务由业主继承"，物业无权为业主设定义务。第五，数据证明所谓"节能效益"的全部资金来源就是业主的热水费——业主为168万投资全额买单，却零收益。第六，节能费不仅未随改造推进下降，反而逐年攀升（2024年比2022年增加16.2%），节能效果越好业主缴费越高，完全悖论。

#### 对"管理规约"的质证

**质证意见**：

### 7.1 事实

- **时间**：2026年1月15日（趁原告上班不在家，强行上门拆除）
- **行为**：物业拆除原告门外热水表，导致热水断供
- **后果**：原告被迫自费安装热水器
- **背景**：原告诉前热水费全部足额缴纳；已立案等待开庭期间
- **非孤立事件**：物业此前已对其他多户拒缴热水费的业主实施了同样的拆表行为

### 7.1a 派出所出警记录（重要新证据）

物业在拆除多户业主热水表的过程中，每次均有业主报警，**派出所均有出警**：

- **民警现场明确告知物业**："你没有依据，不可以这样做"——严厉制止物业的拆除行为
- **物业的反应**：拒绝服从民警制止，态度**极其蛮横**，继续实施拆除
- **证据**：业主持有**两次**出警现场的录像和录音

**法律价值评估**：

| 维度 | 价值 | 说明 |
|------|------|------|
| **公权力背书** | 🔴 极高 | 民警的"没有依据"不是原告的主张，而是**执法机关的现场判断**。虽然不等于法律裁判，但在法庭上的说服力远高于原告单方面指控 |
| **故意违法** | 🔴 极高 | 物业**被执法机关明确警告后仍然继续**——这证明物业的行为不是误判，而是**明知违法故意为之** |
| **系统性侵权** | 🔴 极高 | 不是只拆原告一家，而是**系统性拆除多户**——证明这是物业的惯常手段："不交钱就断供"的胁迫模式 |
| **品格证据** | 🟡 高 | 物业对执法人员态度蛮横、拒绝服从，展示了物业对法律和公权力的漠视。法官在裁量时会考虑当事人的行为模式 |

**庭审使用建议**：

> 这组证据应在举证阶段作为**原告主动举证的重点**（不仅仅是反驳，而是进攻）。建议在法庭辩论收尾阶段呈现——先用人工费和节能费打理性牌，最后用拆表+出警+拒绝执法打情感牌。让法官看到的不只是一个收费争议，而是一个**系统性漠视业主权利、公然抗拒执法**的物业公司。

### 7.2 法律定性（五个角度）

| 角度 | 定性 | 法律依据 |
|------|------|---------|
| 合同违约 | 物业服务合同附件3约定热水温度不低于40度，拆表断供=停止履约 | 物业服务合同 |
| 侵害共有财产 | 若热水表属共用设施组成部分，擅自拆除=侵害业主共有财产 | 《民法典》第274条 |
| 诉讼期间改变现状 | 争议未解决即单方面断供，有以断供要挟业主之嫌 | 诉讼诚信原则 |
| 报复性停止服务 | 原告起诉→物业拆表，时间先后暗示因果关系 | 《民法典》 |
| **抗拒行政执法** | 民警现场制止，物业拒绝服从，继续强行拆除 | 《治安管理处罚法》第50条（阻碍国家机关工作人员执行职务） |

### 7.3 证据战价值

这件事**帮了原告的忙**——它暴露了物业的真实立场：

1. **物业认为自己有权处置热水设备** → 请出示产权证明
2. **物业认为不交费就可以断供** → 说明物业把热水当"商品"而非"物业服务义务"
3. **物业在诉讼期间采取对抗性措施** → 显示物业对业主权利的漠视
4. **派出所民警已经告知物业"没有依据"** → 物业是明知违法仍然故意为之
5. **不是个案而是系统性行为** → 物业的惯常手段就是以断供胁迫业主

## 第八部分：调解策略

### 8.1 绝对不能接受的方案

- 物业只是象征性降价几块钱，不触及产权和经营权问题
- 调解协议中包含"业主不再就此事提起诉讼"的封口条款
- 调解协议中回避产权归属问题

### 8.2 可以接受的条件

- 物业**书面确认**热水设备归业主共有
- 物业**承诺**将热水收费标准提交业主大会表决
- 退还原告全部或大部分费用
- 赔偿热水表被拆和安装热水器的损失
- 物业承诺公开能源管理合同及财务数据

### 8.3 调解方案对照表

| 方案 | 内容 | 是否接受 |
|------|------|---------|
| 最佳 | 物业确认设备共有 + 提交业主大会 + 退费 + 赔偿 | 接受 |
| 可接受 | 退费 + 赔偿 + 物业承诺完善程序 | 考虑接受 |
| 不接受 | 象征性降价 + 封口条款 | 拒绝 |
| 坚决不调解 | 物业态度傲慢、拒绝任何实质让步 | 坚持要求判决 |

**核心原则**：败诉比烂调解更有价值——败诉判决可以上诉，烂调解没有救济途径。

### 一、攻击线一：节能费用 → 倒逼产权认定

#### 1.1 逻辑链逐步拆解与审查

**Step 1：42元/立方米单价中包含"节能费用"**

| 维度 | 评估 |
|------|------|
| 推理 | 物业成本表中有"节能费用"一栏 → 该费用被计入热水费成本 → 由业主承担 |
| 证据 | Excel成本分摊表（物业自己提供）✅ |
| 强度 | **强**。物业自己的账目，无法否认 |
| 漏洞 | 无。这是物业自己的数据 |

**Step 2："节能费用"实为能源管理合同的利益分成**

| 维度 | 评估 |
|------|------|
| 推理 | Excel"水费/节能费"金额 = PDF结算单中乙方实得金额（三年Q1完全吻合） → 此项就是支付给能源公司的分成 |
| 证据 | Excel数据 + PDF结算单，三年完全吻合（49,471.20 / 42,246.70 / 43,348.01）✅ |
| 强度 | **强**。数字完全一致，铁证 |
| 漏洞 | ⚠️ 我方只有Q1数据，全年数据尚未完整对比。对方可能辩称"Q1巧合一致，其他季度不同" |
| **补强** | **提取PDF中全部8个季度的结算数据，逐季对比Excel**。如果8个季度全部吻合，对方无法反驳 |

**Step 3：该利益分成来自能源管理合同**

| 维度 | 评估 |
|------|------|
| 推理 | PDF就是能源管理合同的季度结算单 → 分成依据就是该合同 |
| 证据 | PDF结算单抬头、签章✅；能源管理合同原文✅ |
| 强度 | **强** |
| 漏洞 | 无 |

**Step 4：该合同是物业用热水设备与第三方签订的商业合同**

| 维度 | 评估 |
|------|------|
| 推理 | 合同标的是"热水系统节能改造"→ 改造对象就是热水设备 → 物业以甲方身份签订 |
| 证据 | 能源管理合同原文✅ |
| 强度 | **强** |
| 漏洞 | 无 |

**Step 5：物业签约需具备产权或业主授权**

| 维度 | 评估 |
|------|------|
| 推理 | 处分/经营共有财产需要业主授权（《民法典》第278条、《物业管理条例》第55条）|
| 证据 | 法条本身✅ |
| 强度 | **中强** |
| ⚠️ **核心漏洞** | **这一步的前提是"热水设备属于业主共有财产"——但这恰恰是需要证明的结论**。如果对方主张设备不是共有财产（比如"是物业投资购买的"），则第278条和第55条不适用。这存在**循环论证风险**：用"设备是共有的"推出"物业需要授权"，再用"物业没有授权"推出"设备是共有的" |
| **补强** | 必须**独立证明**设备归业主共有，不能依赖本攻击线自身的逻辑闭环。独立证据：(1) 物业服务合同第31条+附件3（物业自己签字认定"共用设施设备"）；(2) 住宅说明书（开发商建设配套设施）；(3) 物业无产权证明（无购买发票、无产权登记、无转移协议）。**关键**：第31条是最强证据，因为这是物业自己签的——等于物业自己承认了 |

**Step 6：物业无产权且无业主授权**

| 维度 | 评估 |
|------|------|
| 推理 | 物业无法出示产权证明 + 无业主大会决议 → 既无产权又无授权 |
| 证据 | 消极事实（物业无法举证证明有权），举证责任在物业方✅ |
| 强度 | **中** |
| ⚠️ **漏洞** | 对方可能辩称"不需要产权也不需要授权——管理权本身就包含合理范围内的经营决策权"，或者援引管理规约/住户手册中的概括性授权条款 |
| **补强** | 需要提前审阅**管理规约PDF**全文，查看是否有任何可以被解读为"概括授权物业进行热水经营"的条款，如果有则需准备反驳 |

**Step 7：利益分成不应计入业主成本 → 42元应降低 → 差额即退费金额**

| 维度 | 评估 |
|------|------|
| 推理 | 合同缺乏法律基础 → 分成不应由业主承担 → 从成本中扣除 → 单价应降低 → 多收部分应退还 |
| 证据 | 需要精确计算扣除节能费后的合理单价，与42元对比 |
| ⚠️ **漏洞** | **退费金额计算逻辑不严密**。目前的6,083元退费请求的计算依据尚未完整验证。如果扣除节能费后，重新计算的合理单价仍然接近42元（因为物业确实亏损），法官可能认为"实际没多收" |
| **补强** | **必须完成退费金额的精确计算**：(1) 原告实际缴费总额；(2) 扣除节能费和重复人工费后的合理成本；(3) 按合理成本重新计算的应缴金额；(4) 差额即退费金额。数字必须经得起对方律师的逐项核对 |

#### 1.2 法官可能"下车"的三个关键节点

**下车点A：Step 5 — "是否需要触及产权"**

法官最可能在这里停下来，说："本案是物业服务合同纠纷，我审查收费是否合理就行，不需要对设备产权作出认定。"

- **应对**：强调产权是审查成本合理性的**必经前提**，不是独立的诉讼请求。使用已准备的话术（6.3节），给法官"只需援引合同条款"的低门槛台阶
- **风险等级**：🔴 高 — 这是最可能、也是最危险的下车点

**下车点B：Step 7 — "扣不扣节能费，42元还是亏"**

即使法官认同节能费不应计入，但看到物业即使扣除节能费仍然亏损（因为还有其他合理成本），可能认为"实质上没有多收"。

- **应对**：(1) 必须同时打人工费重复收取（攻击线二），把两项都扣掉后差异就很大了；(2) 强调法律层面的违法性独立于经济层面的盈亏——"亏损不是违法的理由"
- **风险等级**：🟡 中高

**下车点C：Step 6 — "管理权隐含经营决策权"**

法官可能认为物业的管理权本身就包含"合理范围内的经营管理决策权"，15年合同虽然时间长但"本质是为了节能降耗"。

- **应对**：(1) "两难困境"论证——如果是管理则不应经营，如果是经营则缺乏全部法律要件；(2) 强调168万投资+15年+利益分成的规模远超"日常管理"范畴
- **风险等级**：🟡 中

#### 1.3 对方律师可能的反驳角度

**反驳1："物业服务合同第31条说的是'维护保养义务'，不是产权认定"**

对方会说：合同写"共用设施设备"只是界定物业的维护保养范围，不是对产权的法律认定。产权应以房产登记、建设规划许可证为准。

- **杀伤力**：🔴 高。这是最精准的法律反驳——合同条款的确不等于产权登记
- **应对**：(1) 反问对方出示产权证明——物业方自己也没有产权登记；(2) 在双方都无产权登记的情况下，物业自己签字确认的"共用设施设备"认定就是最有力的间接证据；(3) 结合住宅说明书（开发商配建）和《民法典》第274条的"公用设施归业主共有"推定规则
- **💡 关键准备**：查阅是否有类似案例的裁判文书——其他法院是否根据物业服务合同中的"共用设施设备"表述认定了产权归属

**反驳2："节能改造有利于业主，属于善意管理"**

对方可能援引"无因管理"或"商业判断规则"——物业为了业主利益进行节能改造，属于合理的管理行为。

- **杀伤力**：🟡 中。法官可能同情"初衷是好的"
- **应对**：(1) "好意"不能替代"授权"——法律要求的程序不能因为结果好而豁免；(2) 实际上业主未从中获益（收益被物业和第三方分成，业主零收益）；(3) 168万投资+15年分成的规模不是"善意管理"，是商业投资

**反驳3："原告不是适格主体质疑合同效力"**

产权归全体业主共有 → 挑战合同效力应由业委会或全体业主 → 个别业主无权。

- **杀伤力**：🟡 中高。如果法官想回避，这给了很好的程序性理由
- **应对**：(1) 原告不请求确认合同无效，只请求认定"节能费不应计入业主成本"——这是个人退费请求的组成部分；(2) 《民法典》第274条是强制性规定，任何权利被侵害的业主都可以起诉

**反驳4："即使扣除节能费，42元仍然合理（物业在亏损）"**

用成本表证明即使去掉节能费，剩余成本仍超过收入。

- **杀伤力**：🔴 高。如果数字上说得通，法官很可能采纳
- **应对**：**必须同时攻击人工费**（攻击线二），把节能费+人工费两项都扣掉。根据2022年数据：总支出1,112,123 - 节能费168,048 - 人工费344,663 = 599,412元，而应收热水费915,543元 → 扣除两项后不仅不亏损，反而**盈利316,131元**。这个数字是致命的
- **💡 关键数据**：必须在庭审中准确呈现这个计算

物业可能出示2005-2025年的单价变化趋势，证明42元是多年演变的结果，不是物业任意制定。

- **杀伤力**：🟡 中。给法官"42元有历史依据"的印象
- **应对**：(1) 历史定价不等于合法定价——程序是否合法与价格水平无关；(2) 需要分析`历年热水费单价（2005-2025）.xls`，看42元是何时开始的、调价是否经过业主大会

#### 2.2 关键法律概念辨析："物业费包含热水人工成本"的法律依据

这是攻击线二的**命门**。需要三个层面论证：

**层面一：合同约定层面**
- 物业服务合同附件3约定物业负责"供水系统包含热水系统的日常运行保养和维修"
- "日常运行"是否包含"雇佣锅炉工烧水"？→ **这一点有解释空间**
- ⚠️ 需要阅读合同原文的**精确措辞**，这是最关键的证据

**层面二：费用对价层面**
- 业主缴纳物业费 → 物业提供物业服务 → 合同附件3列明的服务包含热水系统 → 物业费是热水服务的对价之一
- 但对方可以反驳：物业费是**管理维护**的对价，热水**运营**是独立服务，住户手册规定另行收费
- ⚠️ 这里的关键区分是**"维护设备"vs"运营服务"**——换水泵修管道是维护，烧水供热是运营

**层面三：行业惯例层面**
- 上海其他小区的热水服务模式是什么？人工费是否通常包含在物业费中？
- ⚠️ 如果行业惯例是"热水运营人工费单独收取"，则我方论证会被削弱
- 🔍 这属于尚未调研的内容

**💡 最稳妥的论证路径**：

即使无法完全证明"物业费已包含热水人工成本"，可以退到以下论证：

> 物业服务合同约定物业负责热水系统的**日常运行**。这一义务是物业费的对价之一。物业在履行这一义务时产生的人工成本，如果要另行收取，必须有明确的合同依据或业主大会决议。住户手册写的是"根据**运营成本**制定单价"——"运营成本"的自然含义是水、电、燃气等**物料消耗**，不包含人工费（人工费是物业自身的运营成本，不是热水服务的运营成本）。

这个论证的好处是：不需要证明"物业费一定包含人工"，只需要证明"热水费不应该包含人工"。

#### 2.3 对方律师可能的反驳角度

**反驳1："物业费不包含热水运营成本，热水是独立的专项服务"**

对方核心论点：物业费覆盖公共管理（保安、保洁、绿化、电梯维护等），热水供应是专项服务，有独立的收费依据，人工费是热水运营成本的一部分。

- **杀伤力**：🔴 高。这是最有力的反驳，也是最可能说服法官的
- **应对**：(1) 物业服务合同附件3写了"供水系统包含热水系统"——这就是物业服务的组成部分，不是"独立的专项服务"；(2) 反问对方：如果热水是独立服务，那就应该有独立的服务合同、服务标准、定价程序——这些统统没有；(3) 住户手册写的是"运营成本"，不是"全部成本"——人工费属于物业的管理成本，不属于热水的运营成本

**反驳2："住户手册已经明确授权物业按成本定价，成本包含人工"**

住户手册写了"热水费单价由物业根据运行成本制定"→"运行成本"当然包含人工。

- **杀伤力**：🟡 中。取决于法官如何解读"运行成本"
- **应对**：(1) 住户手册由物业单方制定，按《民法典》第498条格式条款规则，"运行成本"应作不利于制定方的解释；(2) "运行成本"的通常文义是水、电、气等直接物料消耗，不包含人力成本——人工费是物业自身的履约成本，不是热水的运行成本；(3) 即使"运行成本"包含人工，定价权≠重复收取权——同一笔人工费已由物业费覆盖，不能在热水费中再收一次；(4) 备用论点：住户手册根本不是合同（前言："管理处保留解释及修改条文的权利"，不满足《民法典》第469条合同成立要件）

**反驳3："人工费金额合理，有锅炉工薪资证明"**

物业提供锅炉工薪资社保记录，证明人工费是真实支出。

- **杀伤力**：🟡 中。证明"真实支出"不等于"应由业主承担"
- **应对**：(1) 真实≠合理≠合法——即使每一分钱都是真实支出，问题是"该不该由业主出"；(2) 需要分析锅炉工薪资Excel文件：人数是否合理？薪资水平是否合理？是否有"吃空饷"现象？
- **💡 证据缺口**：**三份锅炉工薪资Excel尚未分析**

**反驳4："8,000元/季是小数目，与人工成本~9万/季不是同一笔"**

8,000元是能源公司合同约定的固定补贴，与热水运营的实际人工成本无关。

- **杀伤力**：🟢 低。金额虽小但暴露了机制问题
- **应对**：(1) PDF原文写的就是"扣除人工费"，不是"补贴"；(2) 即使金额小，也证明物业从多个渠道获取人工费收入——向业主收一次、从能源公司收一次；(3) 这8,000元有没有从热水成本中扣减？如果没有（Excel数据显示确实没有），就是额外收入

### 三、两线配合的整体风险评估

#### 3.1 三种场景推演

**🟢 最佳场景（概率~15%）**

法官认同攻击线一的完整逻辑：热水设备属于共用设施设备 → 能源管理合同未经业主授权无效 → 节能费不应由业主承担。同时认同攻击线二：人工费构成重复收费。判决退还全部6,083元 + 赔偿热水表拆除损失。判决书说理中认定热水系统为共用设施设备。

> 触发条件：法官有魄力、证据链完整、对方辩护无力

**🟡 最可能场景（概率~50%）**

法官回避产权问题（攻击线一的Step 5被"下车"），但认同人工费存在问题（攻击线二部分成立）。判决物业退还部分费用（30-50%），要求物业完善收费程序。判决书不触及产权，但可能在说理中提及"成本构成存在瑕疵"。

> 触发条件：法官保守、倾向和稀泥、认为产权问题"太大"不适合在小标的案件中认定

**🔴 最差场景（概率~25%）**

法官认为42元/立方米总体合理（即使程序有瑕疵），驳回退费请求。或者以案由不当驳回起诉。攻击线二的Step 2被击破（法官认为物业费不包含热水人工成本）。

> 触发条件：对方成功论证"热水运营是独立专项服务"、物业费与热水人工成本无关

#### 3.2 两线配合的关键数据武器

**必须在庭审中呈现的计算：**

```
2022年总支出: 1,112,123元
- 扣除节能费（利益分成）: -168,048元
- 扣除人工费（重复收取）: -344,663元
= 合理运营成本: 599,412元

2022年应收热水费: 915,543元
- 合理运营成本: 599,412元
= 盈余: +316,131元（而非物业声称的亏损196,580元）
```

**这组数字的价值**：彻底粉碎物业"42元还在亏损"的抗辩。扣除两个争议项目后，物业不仅不亏损，反而每年盈利31.6万元。

⚠️ **但前提是两条线都被法官采纳**。如果只采纳一条：
- 只扣节能费：1,112,123 - 168,048 = 944,075元 > 915,543元 → 仍然亏损2.9万元 → 物业"亏损"抗辩仍然成立
- 只扣人工费：1,112,123 - 344,663 = 767,460元 < 915,543元 → 盈利14.8万元 → 物业"亏损"被打破

**💡 关键发现：攻击线二（人工费）的数额杀伤力远大于攻击线一（节能费）**。人工费占成本31-37%，节能费占15-18%。单独扣除人工费即可打破亏损叙述，单独扣除节能费则不够。

→ **这意味着攻击线二不只是"稳"，更是"必须拿下"的一条线**。

#### 3.3 被告成本数据的三大内部矛盾（来自分支报告，可直接用于庭审质证）

**矛盾一：德同报告社保数据是虚构的**
- 个税申报表（法定文件）显示王文乾、马春生两年全程专项扣除=0，从未缴纳社保
- 德同评估报告却将三人的社保（1,541.67元/月）和公积金（1,600.16元/月）整体列入热水成本
- 被告向业主收取了从未实际支出的社保费用

**矛盾二：人工成本是固定预算摊销，非实际发放**
- 2022年每月人工成本精确为28,721.95元（12个月完全相同，精确到分）
- 2023年每月精确为30,628.60元（同上）
- 实际工资受年终奖、请假、离职等影响，不可能12个月完全一致
- 庭审建议：要求被告出示工资银行发放流水，与成本表数字对比

**矛盾三：2022年涨价悖论（最具质证杀伤力）**
- 2021→2022年：热水单价从39元涨至42元（+7.7%）

- 庭审话术：*"如果涨价的原因是运行成本上升，请解释为什么人工成本在涨价的同一年下降了44,506元？"*

| 时期 | Excel"节能费用" = | 性质 |
|------|-------------------|------|
| 2020-2021年 | 总节能费 × 甲方（物业）比例 | 物业自己的**收益**，却以"费用"名义列示 |
| 2022-2024年 | 总节能费 × 60% − 8,000元 | 物业对乙方的**支出**，直接作为成本转嫁业主 |

两种口径并存于同一张成本表，无论哪种对业主都不利。2020-21年把收入伪装成费用；2022-24年把商业支出转嫁业主。

**本章节总结**：

攻击线二（人工费重复）看似"稳"，实则有一个**核心命门**：必须证明"物业费已覆盖/应覆盖热水运营人工成本"。这一步的论证如果不扎实，整条线会塌。**最关键的证据**是物业服务合同的原文——附件3的"日常运行"措辞是否足以涵盖锅炉工人工成本。

攻击线一（节能费→产权）的逻辑链整体较长，法官有多个"下车点"，但Step 1-4（证明节能费就是利益分成）的证据链非常扎实。关键风险在Step 5（是否需要触及产权）。

**两条线的最佳配合方式**：先出攻击线二（人工费），建立"物业多收了"的事实基础，迫使法官必须审查成本构成；再出攻击线一（节能费），在法官已经开始审查成本的框架下，追问"节能费是什么"→ 自然引出产权问题。

> 
> 第一，关于人工费重复收取。物业服务合同附件3明确约定，被告负责供水系统包含热水系统的日常运行保养和维修。这些人工成本应当包含在物业费中。但被告在热水费中再次收取锅炉工薪资，占热水费收入的37%至41%，构成重复收费。这个问题与设备产权无关——同一笔人工费不能收两次。

> 第二，关于节能收益分成。被告的成本明细中包含了来自能源管理合同的利益分成。这份15年期、168万元投资的商业合同，是被告未经业主授权、与第三方签订的。合同产生的成本和分成不应由业主承担。
> 
> 第三，关于定价程序。该收费标准从未经过业主大会表决，被告单方面制定并执行，违反《民法典》第278条。
> 
> 第四，在本案诉讼期间，被告擅自拆除了原告门外的热水表，导致热水断供。原告诉前热水费全部足额缴纳。而被告自己签署的两份合同证明——原始设备归业主共有，新改造设备归能源公司——被告什么都不拥有，凭什么拆除？

### 10.3 第三阶段：法庭辩论

**辩论策略说明**：先出稳牌（人工费重复收取，不依赖产权认定），再出重牌（节能费→产权倒逼），再出杀手锏（两难困境+三方产权悖论），最后用热水表事件收尾。

**第一轮（主攻人工费重复收取——最稳的攻击线，独立于产权问题）**：
> "审判长，我首先就人工费重复收取问题向法庭陈述。
> 
> 物业服务合同附件3明确约定：被告负责'供水系统包含热水系统的日常运行保养和维修'。这是被告的服务义务，人工成本应当包含在物业费中。但被告在热水费成本中又单独列出人工成本——2022年34.5万元，2023年36.8万元，2024年1-9月就已达30.7万元，占热水费收入的37%到41%。
> 
> 也就是说，同一批锅炉工的同一份工资，业主通过物业费交了一次，又通过热水费交了第二次。
> 
> 更严重的是，根据能源管理合同的季度结算单，能源公司每季度还要从分成中扣除8,000元'人工费'返还给被告——这笔返还并未从热水成本表的人工成本中扣减，说明被告通过两个渠道获取热水人工费：热水费中向业主收取，同时从能源公司扣回。这构成重复收费。
> 
> 这个问题与热水设备产权归属无关——无论设备归谁，同一笔人工费不能收两次。"

**第二轮（主攻节能费→引出产权问题）**：
> "被告的成本明细中，'节能费用'实际是支付给第三方能源公司的利益分成——我方通过对比PDF结算单和Excel成本表，已经证明两者金额完全一致。这笔分成来自一份15年的能源管理合同。请问被告：签这份合同时，有业主大会的授权吗？用的设备是谁的？如果设备归业主共有——物业服务合同第31条和附件3已经写明了——被告有什么权利用业主的设备签商业合同？"

**第三轮（两难困境——核心武器）**：
> "请被告回答一个基本问题：热水服务是物业服务还是商业经营？
> 
> 如果是物业服务——物业服务合同附件3不仅约定被告负责热水系统维护，还明确要求'保证热水温度不低于40度'。这是结果义务，意味着整个热水服务都在合同范围内。合同第六条采用包干制，物业费就是履行全部合同义务的对价。那么人工成本应当包含在物业费中，定价也需要经业主大会表决。
> 
> 如果是商业经营——经营许可在哪里？与业主的服务合同在哪里？价格备案在哪里？业主授权在哪里？被告享受了经营的全部权利，却未履行经营的任何义务。"

**第四轮（三方产权悖论——用物业自己的合同打脸）**：
> "被告如果主张热水设备归其所有，我请法庭注意被告自己签署的两份合同之间的矛盾：
> 
> 第一份，物业服务合同第31条，被告自己签字确认热水系统是'共用设施设备'——按照《民法典》第274条，这归业主共有。
> 
> 第二份，能源管理合同，约定改造后的新设备在合同期内（15年）归乙方即能源公司所有。
> 
> 也就是说：原始设备，被告自己的合同说是业主的；新改造设备，被告自己的合同说是能源公司的。被告两头都不沾——那被告凭什么主张拥有这些设备？凭什么拆除原告门外的热水表？"

**第五轮（回应"亏损"抗辩+热水表事件收尾）**：
> "被告声称42元还在亏损。但亏损不是违法的理由。被告的成本表中，利益分成和人工费两项合计超过总支出的56%。用包含违法成本项目的报表证明亏损，再用亏损论证违法成本合理——这是循环论证。所有导致亏损的决策——签约、改造、定价——都是被告单方面做的，后果不应由业主承担。
> 
> 最后，关于被告在诉讼期间拆除原告热水表一事。被告拆除的热水表，究竟是谁的财产？如果是共用设施，被告无权拆除。如果被告认为是自己的，请出示产权证明——但正如我刚才所述，被告自己的两份合同已经证明，被告什么都不拥有。原告诉前热水费全额缴清，被告没有催缴理由——拆表不是催缴，是惩罚维权的业主。"
