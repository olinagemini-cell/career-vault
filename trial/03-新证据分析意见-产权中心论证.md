- 证据14-金山房管公众号文章（《物业费，一篇文章讲清楚——基础服务、增值服务、特约服务》）

## 一、新证据的核心增量

### 增量1：能源管理合同第7.3条——被告不存在人工成本（致命新发现）

补充证据清单中，**证据8《能源管理合同》**新增了一项证明目的：

> "合同7.3条约定，乙方上海济通中普能源科技发展有限公司负责设备的日常管理，说明被告不存在人工成本。"

**分析**：这是此前分析中**完全遗漏**的关键条款。此前我们一直围绕"人工费不应计入运行成本"进行论证，但如果合同第7.3条明确约定设备日常管理由**乙方（能源公司）**负责，那么被告在成本表中列支的锅炉工薪资（年化34-36万元）不仅是"不应计入"的问题——被告声称的这些人工成本**根本就不应存在**。所谓的"专职锅炉工"岗位是一个从头虚构的成本项目。

**对攻击线的影响**：攻击线二（人工费）从"不应计入"升级为"虚构成本"——杀伤力质变。

### 增量2：证据12——业委会公开信（新增证据，此前未掌握）

补充证据清单**证据12——业委会《致全体业主的公开信》**是一个全新的、此前策略文档中未涉及的重磅证据。关键内容：

1. **"本小区没有配套的煤气供应系统，并且在电力设计时未考虑安装电热水器的容量，每户只有4KW的入户容量，无法支持安装电热水器，因此，小区的锅炉热水系统是居民生活用热水的唯一基础配套设施。"**

2. **"基础服务不是为单个业主提供的，而是为整个物业区域内的所有受益人服务的，满足业主对安全、清洁、有序、设施正常运行等最基本的生活和工作需求。因此，案涉的热水系统并非增值服务，而是小区的基础服务。"**

3. **"被告定义的热水费的成本由水费、电费、燃气费、能源费、设施设备更新维修费和人工费等相关费用组成，明显存在重复收费、多收费的问题。"**

**分析**：业委会作为全体业主的法定代表机构，其公开信具有三重价值：

- **事实认定价值**：业委会确认了小区无燃气、电力容量不足以支持电热水器的客观事实——热水锅炉是唯一热源。这意味着热水服务不是"可选的"增值项目，而是**生存必需的基础设施**。
- **法律定性价值**：业委会明确将热水系统定性为"基础服务"而非"增值服务"——这与被告可能的"增值服务/独立服务"抗辩直接矛盾。
- **立场印证价值**：业委会明确认为被告的收费"明显存在重复收费、多收费的问题"——原告的诉讼主张与业主自治机构的判断完全一致，而非个人"过度维权"。

### 增量3：证据14——金山房管公众号文章（法律定性框架）

金山房管公众号文章（上海市金山区住房保障和房屋管理局官方公众号，发表于2025年10月20日）系统阐述了物业管理中**三类服务的法律边界**：

| 服务类型 | 定义 | 服务对象 | 收费方式 | 合同依据 | 与本案的对应 |
|---------|------|---------|---------|---------|------------|
| **基础服务** | 物业合同约定的基本服务（设备设施维保、保洁、绿化、安保） | 全体业主 | 物业费覆盖 | 物业服务合同 | **本案热水服务应属此类** |
| **增值服务** | 物业利用自身资源为提升品质、增强竞争力而提供的超出基础范畴但面向全体/部分业主的服务 | 全体或部分业主 | 物业费外可另收，但需明确约定 | 物业服务合同+补充约定 | 被告可能主张此类 → 但仍需合同依据 |
| **特约服务** | 物业合同约定范围**之外**，根据特定业主个性化需求量身定制的非标准服务 | 特定业主（个人或群体） | **谁使用谁付费**，需签特别协议 | 独立的特约服务协议 | 本案热水明显不属此类 |

**文章的核心逻辑链**：
1. 基础服务 = 物业合同约定 = 物业费覆盖 → 业主基本权利
2. 增值服务 = 提升品质 → 可另收费但需明确约定在合同中
3. 特约服务 = 合同外个性化 → 需独立协议 + 谁用谁付费

**对本案的关键适用**：

本小区热水系统具有以下特征：
- 无煤气供应 + 电力容量仅4KW/户 → 锅炉热水是**唯一热源**（证据12业委会确认）
- 全小区1,046户**全部依赖**该热水系统（证据12、13）
- 热水系统列入物业合同附件三"共用设施设备清单"（原-1）
- 被告的《住宅说明书》（原-12）和《住户手册》（原-10）均记载热水为建筑配套设施

按照金山房管公众号的权威分类标准，**唯一热源 + 全体业主依赖 + 物业合同明确约定 + 建筑配套设施 = 基础服务**。基础服务的成本应当由物业费覆盖（尤其在包干制下），被告在热水费中另行收取人工费和商业合成分成缺乏法律依据。

## 二、新证据如何重构三条攻击线

### 攻击线一（节能费/产权）——从"倒逼"到"正面确权"

**原有策略**：通过质疑"节能费是否应由业主承担"倒逼法官审查产权。缺点是法官有多个"下车点"，可能回避产权问题。

**新证据强化后**：

1. **业委会公开信（证据12）确认**：热水是"唯一基础配套设施" → 既然是唯一基础配套，设备必然属于业主共有（《民法典》第274条）
2. **住宅说明书（证据11）确认**：热水设备是"公共设备"，"业主只需要承担热水供应成本，人工费用应包含在物业费中"——这是开发商在竣工文件中的原始定性
3. **金山房管分类框架（证据14）**：基础服务的设备和成本归属有明确的法律逻辑

**策略调整**：不再"倒逼"产权，而是**正面建立产权归属**。证据链：
- 住宅说明书（开发商原始文件）→ 热水是建筑配套设施
- 物业服务合同附件三 → 被告自认"共用设施设备"
- 业委会公开信 → 业主自治机构的正式定性
- 金山房管公众号 → 行政主管部门的权威分类标准
- 《民法典》第274条 → 法律强制规定

五层证据共同指向一个结论：**热水设备属于业主共有财产，被告只是受托管理者，无权未经授权进行商业经营。**

### 攻击线二（人工费）——从"不应计入"到"虚构成本"

**原有策略**：论证人工费不属于"运行成本"，不应由业主承担。

**新证据强化后**：

1. **能源管理合同第7.3条（证据8新增）**：乙方负责设备日常管理 → 被告声称的"必须雇佣专职锅炉工"是一个**伪命题**——合同已约定由能源公司负责运营，被告根本不需要锅炉工岗位
2. **业委会公开信（证据12）**：明确认定被告收费"明显存在重复收费、多收费"
3. **住宅说明书（证据11）**：开发商原始文件写明"人工费用应包含在物业费中"
4. **金山房管分类（证据14）**：基础服务的人工成本属于物业费法定构成（《物业服务收费管理办法》第11条）

**策略调整**：人工费不仅是"不应计入运行成本"的问题，而是：
- 能源管理合同第7.3条已约定由乙方负责日常管理 → 被告的"锅炉工"岗位是从头虚构的
- 零社保、月度金额固定、"1人算2人"等六大矛盾 → 印证了该岗位确实不存在真实的专职员工
- 如果被告确实雇佣了人员，那这些人员是在履行能源管理合同第7.3条下本应由能源公司承担的义务 → 费用应向能源公司追偿，而非转嫁业主

### 攻击线三（定价程序）——证据14加固了"协商定价"的法理基础

金山房管公众号明确指出，特约服务需"签订特别协议"、增值服务需"在物业合同中明确约定"。这一框架反过来证明了：即便是独立于基础服务的收费项目，也必须具备明确的合同依据和双方合意。被告42元/m³的定价既无合同依据，也无协商记录——不符合任何一类服务的定价程序要求。

## 三、以"产权"为中心的证据体系

### 产权确立的五层证据

| 层次 | 证据 | 来源 | 证明力 |
|------|------|------|--------|
| 第一层：开发商原始文件 | 住宅说明书第三章（证据11） | 开发商2004年竣工法定文件 | **最强**：开发商的原始建设文件，证明设备是建筑配套设施 |
| 第二层：物业自认 | 物业服务合同附件三（原-1） | 被告签字盖章的合同 | **极强**：被告自己认定热水系统为"共用设施设备" |
| 第三层：物业自认 | 住户手册第四章（证据9） | 被告制作发放 | **强**：主动援引住宅说明书，印证配套设施定位 |
| 第四层：业主自治机构 | 业委会公开信（证据12） | 业委会（业主法定代表） | **极强**：正式书面认定"唯一基础配套设施" |
| 第五层：行政主管机关 | 金山房管公众号文章（证据14） | 金山区住房保障和房屋管理局 | **权威**：行政主管部门对服务分类的官方解读 |

### 产权确立后的法律后果

一旦法院认定热水设备归业主共有，将产生以下连锁法律效果：

1. **能源管理合同对业主不发生效力**（《民法典》第171条无权代理）
2. **物业的甲方分成收入（年化约14万）属于业主共有财产收益**（《民法典》第282条）
3. **物业将乙方分成转嫁业主属于侵占共有财产收益**
4. **被告拆除热水表的行为从"合同违约"升格为"侵害共有财产"**（《民法典》第274条）

## 四、对被告可能抗辩的预判与应对

### 抗辩1："设备是开发商移交给物业的，产权已随移交发生转移"

**应对**：
- 请被告出示产权转移协议或购买发票（不可能存在）
- 开发商将配套设施移交物业，移交的是**管理权**而非**所有权**
- 《民法典》第274条直接规定建筑区划内的公用设施属于业主共有——这是法定归属，不需要约定，也不能通过约定排除
- 住宅说明书（开发商原始文件）将设备定性为"公共设备"——如果开发商意图转移产权给物业，不会使用"公共"二字

### 抗辩2："金山房管公众号文章不是正式法规文件，不具有法律约束力"

**应对**：
- 该文章的作用不是作为"法规"引用，而是作为**行政主管部门对现行法律适用的官方解读**——帮助法庭准确理解《民法典》第274条、《物业服务收费管理办法》第11条、《物业管理条例》第55条在实践中的适用标准
- 文章将基础服务、增值服务、特约服务的边界进行了系统梳理，该标准来源于现行法律法规，文章本身只是整合说明
- 类似于最高人民法院的理解与适用书籍——不是法律渊源，但具有参考价值

### 抗辩3："业委会公开信只是业委会单方观点，不具法律效力"

**应对**：
- 业委会是全体业主的法定代表机构（《民法典》第277条、《物业管理条例》第15条）
- 业委会对"热水是唯一基础配套设施"的事实陈述（无煤气、电力容量不足）属于客观事实，不受"观点"争议
- 业委会不是本案当事人，不存在"原告串通"的可能性
- 被告作为物业服务企业，其观点与业主自治机构的观点相左时，应以业主大会/业委会的意见为准——这正是"业主自治"法律原则的体现

## 五、代理词重构建议

### 核心调整：从"三线并列"改为"产权主轴 + 三线辐辏"

**原有结构（v3.0策略文档）**：三条攻击线相对独立，各有侧重，配合使用。

**新结构**：以"热水设备产权归业主共有"为论证主轴，三条攻击线作为产权确立后的法律后果依次展开。

### 新的论证逻辑链

```
第一步：确立产权（正面出击，不再迂回）
    五层证据 → 热水设备属于业主共有的共用设施设备

第二步：产权确立后的第一法律后果 → 人工费问题（攻击线二升级版）
    设备是业主的 → 被告受托管理 → 人工费是被告的履约成本 → 属于物业费法定构成
    + 能源管理合同第7.3条 → 被告声称的人工成本岗位根本不应存在

第三步：产权确立后的第二法律后果 → 节能费/能源合同问题（攻击线一升级版）
    设备是业主的 → 被告无权未经授权签商业合同 → 合同对业主不发生效力
    → 合同产生的分成不应由业主承担 → 被告的甲方分成收入属于业主共有财产收益

第四步：产权确立后的第三法律后果 → 定价程序问题（攻击线三升级版）
    设备是业主的 → 基本服务 → 使用基本服务的收费标准需协商确定
    → 被告单方定价违反法定程序

第五步：产权确立后的第四法律后果 → 拆表行为
    设备是业主的 → 被告拆除热水表 = 毁损业主共有财产 + 合同违约 + 对抗执法
```

### 金山房管分类框架的嵌入位置

在第一步"确立产权"中，将金山房管公众号的三类服务分类作为**定性分析框架**嵌入：

- 本小区热水是"唯一基础配套设施"（业委会确认）→ 属于基础服务
- 基础服务的设备是共用设施设备（物业合同自认）→ 属于业主共有
- 基础服务的成本由物业费覆盖（包干制 + 《物业服务收费管理办法》第11条）

## 六、新证据的风险提示

1. **证据12（业委会公开信）的双刃剑效应**：业委会在公开信中可能也提到了物业费的某些合理性，需要通读全文确认是否包含对被告有利的表述。目前仅凭补充证据清单的摘要判断，建议庭审前通读公开信全文。

2. **证据14（金山房管公众号）的参考效力限制**：该文章不是正式法规或规范性文件，法院是否采信其分类标准存在不确定性。建议作为辅助参考而非核心法律依据，核心依据仍然是《民法典》第274条、第278条、第282条。

3. **能源管理合同第7.3条需要核实原文**：补充证据清单中对该条款的表述是概括性的。建议提取合同第7.3条的原文措辞，确认"日常管理"的准确含义和范围——如果仅指设备技术维护而非人员运营管理，则证明力会打折扣。

4. **证据编号已变更**：补充证据清单采用了新的证据编号（8-21），与原有证据清单（原-1至原-14）不同。代理词中需统一使用新的编号体系。

## 七、结论

三份新证据（能源管理合同第7.3条、业委会公开信、金山房管公众号文章）共同指向一个结论：**热水设备是全体业主的共有财产，热水服务是小区的基础服务而非增值/特约/商业经营。** 这一结论将本案的争议焦点从"收费是否合理"升级为"被告是否侵占了业主的共有财产及其收益"。

代理词应以**产权确立**为论证主轴，将三条攻击线作为产权的法律后果依次展开，以金山房管分类框架为辅助定性工具，业委会公开信为事实和立场背书，能源管理合同第7.3条为对被告数据可信度的致命打击。

**案件定位升级**：从"退6,083元热水费"→"确立1046户业主对热水设备的共有产权 + 追回被侵占的共有财产收益"。
