# 第三章 热水设备的法律性质认定

本章解决的核心问题：热水系统及相关设备，法律上属于小区共用设施设备，还是物业自有经营资产？

## 3.1 合同文本的明确认定

物业服务合同对热水设备性质有明确约定，无需推断：

+--------------------------------------------------------------------------------------+
| **物业服务合同原文依据（两处相互印证）**                                             |
|                                                                                      |
| 第十一条第(一)款：「物业共用设施设备的日常运行和维护，详见附件三。」                 |
|                                                                                      |
| 附件三标题：「物业共用部位、共用设施设备的日常运行、保养和维修服务」                 |
|                                                                                      |
| 附件三热水系统栏：「每年保养一次，保证二次供水正常，热水温度不低于40度，泵房整洁。」 |
|                                                                                      |
| 第三十一条第(六)款：共用设施设备定义明确包括「供水管道......水箱、水泵」             |
+======================================================================================+
+--------------------------------------------------------------------------------------+

上述条款证明：物业服务合同已将热水系统（含锅炉、热泵、水箱、水泵、管道）纳入「共用设施设备」范畴，并将其日常运行纳入物业服务内容。

## 3.2 能源管理合同的设备所有权条款

能源管理合同第九节对节能改造后的设备所有权有明确约定：

+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **能源管理合同第九节原文**                                                                                                                                                                                    |
|                                                                                                                                                                                                               |
| 9.1：「在本合同有效期满和甲方付清全部节能收益款项之前，项目（包括设备和设施）的所有权属于乙方（计通中普）。甲方在本合同有效期满后15日内，按规定付清乙方应得全部节能收益款项之后，才有权享有该项目的所有权。」 |
|                                                                                                                                                                                                               |
| 9.3：「节能收益分享期结束，甲方付清乙方全部节能收益款后，节能项目设备及节能系统所有权和所有相关全部资料无偿归甲方所有。」                                                                                     |
|                                                                                                                                                                                                               |
| 合同有效期：效益分享期2015.1.1-2029.12.31，合同现仍在有效期内。                                                                                                                                               |
+===============================================================================================================================================================================================================+
+---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

这一条款产生两个重要推论：

-   节能改造后的热水设备，在合同有效期内（至2029年）所有权属于计通中普，而非物业公司。物业声称「热水设备是物业所有的经营资产」，与合同原文直接矛盾。

-   物业服务合同将热水系统列为共用设施设备，能源管理合同又将节能改造设备所有权归属于第三方节能公司------物业既无设备所有权，又无合同授权将热水运营定性为「增值服务」。

## 3.3 共用设施设备的法律推定

根据《上海市住宅物业管理规定》及相关法规，热水供应系统具备以下共用设施设备的法律特征：

-   服务对象：为小区全体业主提供热水，而非特定个体

-   建设来源：热水系统（常压燃气锅炉+屋顶热水箱+全楼管网）由开发商上海荣联房地产有限公司于2004年建设竣工时作为大楼配套设施建造完成，早于物业公司进驻，不可能属于物业公司自有资产（证据：《圣天地公寓住宅说明书》第三章给水系统第3.1条）

-   开发商原始文件佐证：住宅说明书第三章明确记载「热水由常压燃气锅炉提供热源经屋顶热水箱供热水」，物业公司在其自行编制的住户手册中直接引用同一描述，并注明「具体情况见《新建住宅使用说明书》」，说明物业公司自己也认可该系统属于大楼原有配套设施

-   不可分割性：热水管网属于小区整体供水系统的组成部分

-   物业合同认可：物业服务合同附件三明确将热水系统纳入共用设施设备日常运行范畴

  ------------------------------------------------------------------------------------------------
  **法律推定结论：热水设备属于小区共用设施设备，全体业主共有，物业无权将其定性为自有经营资产。**

## 3.4 「共用设施设备」与「公用设施」的法律等同性及驳斥预案

本节专门应对被告可能以合同措辞差异为由，挑战《民法典》第274条适用性的辩护策略。

### 3.4.1 两个条文的功能定位

  ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **条文**                   **功能**                     **原文要点**
  -------------------------- ---------------------------- --------------------------------------------------------------------------------------------------------------------------------------------
  第271条（权利基础）        说明业主对共有部分享有权利   「业主对专有部分以外的共有部分享有共有和共同管理的权利。」------确立权利基础，但未列举哪些属于共有部分。

  第274条（具体列举）★主力   直接认定公用设施归业主共有   「建筑区划内的其他公共场所、公用设施和物业服务用房，属于业主共有。」------具体列举条款，热水系统等公用设施依此直接归属业主共有，无需推断。
  ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

  ------------------------------------------------------------------------------------------------------------------------------------------
  **引用建议：第274条为主力条文，直接认定热水设备属于「公用设施」归业主共有；第271条为配合条文，说明业主基于此享有共有和共同管理的权利。**
  ------------------------------------------------------------------------------------------------------------------------------------------

  ------------------------------------------------------------------------------------------------------------------------------------------

### 3.4.2 被告的可能反驳及逐条驳斥

被告可能主张：物业服务合同中写的是「共用设施设备」，而非「公用设施」，试图与民法典第274条的表述产生区隔，否定业主共有权。以下三条反驳均不成立：

+---------------------------------------------------------------------------------------------------------------------------------------+
| **反驳一：「共用」本身就等于第274条的「公用」**                                                                                       |
|                                                                                                                                       |
| 《物业管理条例》第55条使用的正是「共用设施设备」这一表述，其含义与《民法典》第274条的「公用设施」完全对应。                           |
|                                                                                                                                       |
| 司法实践中，从未有法院以这两个字（「共用」vs「公用」）的表述差异，否定业主对共有部分的权利。                                          |
|                                                                                                                                       |
| 被告自己签字认定这些设备是「共用设施设备」，恰恰证明被告知晓这些设备供全体业主共同使用------这正是第274条所规定的业主共有的判断标准。 |
+=======================================================================================================================================+
+---------------------------------------------------------------------------------------------------------------------------------------+

+----------------------------------------------------------------------------------------------------------------------------+
| **反驳二：格式条款不利解释原则（《民法典》第498条）**                                                                      |
|                                                                                                                            |
| 合同条款产生歧义时，依据《民法典》第498条，应作出不利于提供格式条款一方的解释。                                            |
|                                                                                                                            |
| 物业是合同的起草方。若「共用设施设备」能被解读为「不归业主共有」，则对业主不利，应作对业主有利的解释，即认定属于业主共有。 |
|                                                                                                                            |
| 被告不能既以起草方身份制造歧义，又以解释方身份从中获益。                                                                   |
+============================================================================================================================+
+----------------------------------------------------------------------------------------------------------------------------+

+-------------------------------------------------------------------------------------------------+
| **反驳三：第274条适用不依赖合同表述**                                                           |
|                                                                                                 |
| 第274条是强制性规定，热水系统是否属于「公用设施」，由其客观功能决定，不取决于合同里叫什么名字。 |
|                                                                                                 |
| 锅炉、水箱、管道供全楼1,046户业主使用，客观上就是公用设施，第274条当然适用。                    |
|                                                                                                 |
| 合同措辞是合同层面的表述，不能改变法律的强制性认定。                                            |
+=================================================================================================+
+-------------------------------------------------------------------------------------------------+

### 3.4.3 庭审话术（建议原文使用）

> 『\"被告强调合同写的是\'共用设施设备\'而非\'公用设施\'，但《物业管理条例》第55条使用的正是\'共用设施设备\'这一表述，其含义与《民法典》第274条的\'公用设施\'完全对应。更重要的是，被告自己签字认定这些设备是\'共用设施设备\'，这恰恰证明被告知晓这些设备供全体业主共同使用------这正是第274条所规定的业主共有的判断标准。被告不能一方面在合同里承认设备是\'共用的\'，另一方面又主张设备归物业所有。\"』

# 第四章 节能费的合法性分析

节能费是热水成本表中的第二大单项支出，7年合计1,238,418元。本章分析其法律性质及向业主收取的合法性。

## 4.1 节能费的真实性质

从能源管理合同第五节可以清晰确认，热水成本表中的「节能费用」，实质是：

+--------------------------------------------------------------------------------+
| **节能费的真实性质**                                                           |
|                                                                                |
| 物业（甲方）按季度支付给计通中普（乙方）的节能效益分成款。                     |
|                                                                                |
| 收益分享期：2015年1月1日至2029年12月31日，共15年。                             |
|                                                                                |
| 分成比例：前5年甲方30%/乙方70%；中间5年甲方40%/乙方60%；后5年甲方50%/乙方50%。 |
|                                                                                |
| 结算频率：每季度结算一次，由乙方向甲方开具发票。                               |
|                                                                                |
| 这是物业与第三方节能公司之间的纯商业安排，与任何业主无关。                     |

## 4.3 物业的双重获益------最致命的证据

## 4.4 节能费向业主收取的三重违法

### 违法点一：完全缺乏合同授权

物业服务合同、住户手册、管理规约中，均无任何条款将节能改造分成费用列为业主应付费用。以下逐一核查：

-   物业服务合同：无任何关于节能费、节能分成的约定

-   住户手册第四章（2004年11月，物业自制）：将热水费构成描述为「运行成本」，未提及节能费。「运行成本」的文义范围是水费、气费、电费等物料消耗------商业分成不属于运行成本。物业单方扩大「运行成本」内涵以包含节能分成，违反民法典第498条格式条款解释规则（疑义利益归业主）。

-   管理规约第十一条（2017年5月26日业主大会通过）：明确规定「利用物业共有部分、共用设施设备从事经营的，应当在征得相关业主、业主大会和物业服务企业的同意后，按规定办理有关手续，获取的收益按约定处理」------规约本身：①承认热水系统是共有部分/共用设施设备；②经营共有部分须经业主大会同意；③收益须按规约处理。物业引入能源管理合同收取分成，既未征得业主大会同意，又将本应属于业主的收益独享，同时将成本转嫁，与规约规定完全相悖。管理规约不仅不构成物业的授权依据，反而构成对物业行为的明确制约。

-   能源管理合同本身：甲方是物业，乙方是节能公司，业主不是任何一方，对业主不具有约束力

### 违法点一补充：住宅说明书------开发商建设档案确认，\"计量收费\"≠经营授权

**三层证明力分析：**

-   第一层（建设来源）：热水系统建设者是开发商上海荣联房地产有限公司，2004年竣工，早于物业公司进驻。物业无任何出资记录，不可能取得所有权。住宅说明书是开发商交付给每一位购房人的法定文件，其记载的设备状态为交付基准，具有最高证明效力。

-   第二层（\"计量收费\"的文义边界）：原文\"热水表计量收费由物业负责\"，字面含义是委托物业代为按表计量并执行收费。\"负责收费\"是管理职责的委托，不是授权物业：①在物业费之外另立独立收费项目；②自主制定热水单价；③将商业分成计入成本向业主转嫁；④签订15年期商业合同并从中分成。住宅说明书由开发商制作，不具备为物业公司创设超出物业服务合同范围的收费权的效力。

-   第三层（物业自认）：物业自制的住户手册第四章原文主动注明\"具体情况见《新建住宅使用说明书》\"，等于物业承认：热水系统的设备属性和收费依据都来自该说明书。物业不能一方面在手册里援引说明书，另一方面在庭上主张说明书内容对自己无拘束力。

  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **结论：向业主收取节能费，既无合同依据，也无法律依据，属于擅自增加收费项目。管理规约第十一条、住户手册第四章、住宅说明书均不构成物业的授权依据------三份文件反而共同证明：热水系统是开发商建设的共用设施，物业仅为受托管理方，无权将商业分成成本转嫁业主。**
  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

### 违法点二：物业同时是节能收益的受益方，再向业主收取成本款，属双重获益

如4.3节数据所示，物业在向业主收取节能分成款的同时，自身也在从节能收益中获得40%的直接分成。这一安排的结果是：

-   节能改造的收益（节省的能源费用）：物业获得40%，节能公司获得60%，业主获得0%

-   节能改造的成本（支付给节能公司的60%分成）：100%转嫁给业主

-   净效果：业主既没有享受节能收益，又承担了全部节能成本

### 违法点三：节能费逆势攀升，不符合\"成本\"的基本逻辑

节能改造于2014年实施，节能效果应随时间稳定或递增。但热水成本表显示，节能费从2022年起持续上涨，2024年比2022年增加27,180元（+16.2%）。

原因在于：能源管理合同的分成比例是对乙方的付款比例，与节能效果挂钩------节能越多，乙方收取的绝对金额越大。物业将这笔商业成本全部转嫁给业主，意味着节能效果越好，业主缴纳的节能费反而越高，完全违背常识。

# 第六章 物业\"增值服务\"主张的逐条驳斥

物业提出以下三项主张，以论证热水运营属其自主经营活动，业主必须另外付费。本章逐一分析其法律效力。

## 6.1 主张一：\"热水是增值服务，与物业服务合同无关\"

  -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **物业主张**                                                             **法律评估**
  ------------------------------------------------------------------------ ----------------------------------------------------------------------------------------------------------------
  热水供应属于增值服务，不在物业服务合同范围内，业主使用热水须另行付费。   【不成立】物业服务合同第十一条及附件三明确将热水系统纳入「共用设施设备日常运行」。合同原文优于物业的口头说法。

  -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

驳斥要点：

-   「增值服务」在物业服务法律体系中特指合同之外、业主自愿选择的额外服务，如家政、停车等。热水供应不属于此类------它是小区基础生活设施服务。

-   合同附件三热水系统栏明确要求「保证二次供水正常，热水温度不低于40度」，这是持续性结果义务，不是可选服务。

-   物业从未就热水服务单独与业主签订「增值服务协议」，不存在任何书面授权文件。

-   住户手册第四章专门设置「热水供应」章节，在物业自制文件中已将热水作为服务内容向业主告知。

## 6.2 主张二：\"物业有权对热水设备进行改造\"

  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **物业主张**                                                                               **法律评估**
  ------------------------------------------------------------------------------------------ -------------------------------------------------------------------------------------------------------------------------------------
  物业自行委托节能公司对热水设备进行改造（换装空气源热泵），属于物业对自有设备的合法处置。   【不成立】设备所有权至2029年属于节能公司（能源管理合同第9.1条），物业对设备无完全所有权。共用设施设备的重大改造亦须经业主大会程序。

  --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

驳斥要点：

-   能源管理合同第9.1条明确：在2029年付清全部节能款之前，设备所有权属于乙方（计通中普），而非物业。物业对该设备无完全处置权。

-   即便设备归物业所有，热水设备属于共用设施设备，根据《上海市住宅物业管理规定》，对共用设施设备的重大改造须经业主大会（或业主委员会授权）决议，物业无权单方决定。

-   实际上，节能改造发生在2014-2015年，物业未经任何业主大会程序擅自进行，程序本身存在违规。

## 6.3 主张三：\"不付热水费，物业有权拆除热水设备停止供热\"

  ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **物业主张**                                               **法律评估**
  ---------------------------------------------------------- ----------------------------------------------------------------------------------------------------------------------------------------------
  业主拒绝支付热水费，物业有权拆除热水设备或停止热水供应。   【不成立且适得其反】拆除权在能源管理合同中属于节能公司（乙方）对物业（甲方）的权利，物业无权转嫁至业主。且物业服务合同包干制下不得擅自停服。

  ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

驳斥要点：

-   能源管理合同第5.5.2条：「逾期付款超过一个月并经乙方书面催缴仍不支付的，乙方可单方解除本合同，有权拆除节能项目设备......」------这是节能公司对物业的权利，前提是物业拖欠节能公司的款项，与业主热水费无关。

## 8.3 双倍返还条款的适用性

物业服务合同第二十四条原文：「乙方违反本合同的约定，擅自提高收费标准的，对超出标准的部分，业主有权拒绝支付；已经支付的，业主有权要求乙方双倍返还。」

适用分析：

-   节能费从未在任何合同中被列为业主应付项目，物业将其计入热水单价属于「擅自提高收费标准」。

-   原告已实际支付2018-2024年热水费，满足「已经支付」的适用前提。

-   本条适用不要求证明物业主观恶意，客观上违反合同约定即可适用。

# 第九章 被告可能的抗辩及应对

### 抗辩一：节能改造是物业自主投资行为，节能费是合理成本

-   节能改造的投资方是乙方（计通中普），合同第4.1.2条第3款：「节能项目投资：由乙方承担。」物业没有为节能改造出资，因此「投资回收」之说不成立。

-   物业只是引入节能公司，并将节能改造的商业分成支出转嫁给业主，属于无本获利。

### 抗辩二：节能改造降低了业主的热水成本，业主已受益，应支付对价

-   节能改造降低的是燃气费，燃气节省的好处在能源管理合同框架内已分配给了物业（40%）和节能公司（60%），业主未从节能收益中获得任何分成。

-   业主得到的只是热水服务------而这本来就是物业服务合同的义务范围内的事项，无需为物业履行合同义务额外支付对价。

### 抗辩三：业主长期支付热水费，说明对收费标准已默认

-   支付本身不构成对收费合法性的认可。住户手册和热水成本表的详细构成，业主直至2024年才通过业主群获知，诉讼时效应从知情时间起算。

-   物业服务合同第二十四条明确赋予业主要求返还权利，该权利不因过去支付而消灭。

### 抗辩四：物业经营范围包含热水服务，因此属合法经营

-   经营范围是工商登记事项，表示物业有资格经营相关业务，但不等于业主对其热水经营模式及收费项目作出了同意。

-   即便经营范围涵盖热水服务，具体收费内容仍须有合同依据；无合同授权的收费项目，仍属违法收费。

# 第十章 庭审建议

## 10.1 证据提交建议

-   要求物业提供与节能费相关的完整合同（正式版能源管理合同），确认当前在案的是否为最终执行版本。

建议以「两线并行、各自独立」为切入点：
