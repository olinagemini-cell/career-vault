## 本知识库简介

本知识库是 **中国/香港租赁平台 SIT（系统集成测试）** 项目的结构化知识库。它涵盖了三个系统之间的端到端数据流：

```
Quotation Tool (QT，报价工具)  →  Origination（业务发起系统）  →  Odessa Back Office（Odessa 后台系统）
```

本知识库有两个目的：
1. **短期目标**：通过消除甲方与乙方之间的信息不对称，加速当前 SIT 阶段的推进
2. **长期目标**：为未来多系统集成项目提供可复用的框架

## 推荐阅读路径

### 路径 A：新加入项目的乙方团队成员
```
00（本文件）  →  10（FAQ 快速了解）  →  01（产品类型）  →  03（角色与流程）  →  02（业务规则）  →  04 + 05（字段映射）  →  06（操作手册）
```

### 路径 B：乙方测试人员执行测试用例
```
06（操作手册）  →  04 或 05（查找特定字段）  →  02（查看业务规则）  →  11（使用模板记录结果与缺陷）  →  10（查 FAQ）
```

### 路径 C：甲方 PM 审查进度与规划
```
08（项目管理）  →  07（自动化策略）  →  02（审查需要更新的规则）
```

### 路径 D：启动新国家/新系统集成项目
```
09（复用模板）  →  00（调整文档结构）  →  用新内容重建 01-06
```
