# 战略汇报模板（精华版）

---

## 两大汇报类型

| 类型 | 适用场景 | 核心逻辑 |
|------|---------|----------|
| **战略破局型** | 防守/转型/解决危机 | 承认问题→找到病根→做出抉择→绝地反击 |
| **战略规划型** | 进攻/新业务/抢地盘 | 看见机会→定义战场→设计打法→构建能力 |

---

## 方案一：战略破局型汇报

**叙事主线**：Past-Problem-Future（过去→问题→未来）

### PPT结构（10页）

| 页码 | 内容 | 核心工具 |
|------|------|----------|
| P1 | 封面+核心主张 | 一句话讲清"由危转机" |
| P2 | 现状还原 | 趋势折线图+舆情关键词云 |
| P3 | 外部根因分析 | PEST分析/消费者画像对比 |
| P4 | 内部根因分析 | 4P营销组合自检 |
| P5 | 战略抉择Where to Play | 重新定义市场/定位 |
| P6 | 致胜策略How to Win | 3C模型（差异化价值主张） |
| P7 | 关键战役1：产品重塑 | 权益/服务拆解表 |
| P8 | 关键战役2：品牌重建 | Campaign时间轴 |
| P9 | 资源诉求 | 预算/权限/人力需求 |
| P10 | 承诺与里程碑 | 甘特图+核心KPI |

### 话术模板
```
"面对[危机事件]，我们的核心判断是[根因分析]。
我们决定[战略抉择]，具体打法是[致胜策略]。
需要您支持[资源诉求]，我们承诺[里程碑目标]。"
```

## 方案二：战略规划型汇报

**叙事主线**：5步战略级联（愿景→赛道→策略→能力→举措）

### PPT结构（10页）

| 页码 | 内容 | 核心工具 |
|------|------|----------|
| P1 | 封面+愿景 | Winning Aspiration |
| P2 | 市场机会 | TAM/SAM/SOM柱状图 |
| P3 | 竞争格局 | 波特五力/竞品雷达图 |
| P4 | 赛道选择Where to Play | 市场细分矩阵（2x2） |
| P5 | 商业模式How to Win | 商业模式画布/价值链图 |
| P6 | 核心优势 | SWOT分析（重点SO策略） |
| P7 | 能力建设 | 能力地图（Gap分析） |
| P8 | 增长路径 | 4P营销组合（产品+推广） |
| P9 | 财务预测 | 数据漏斗模型/ROI预测 |
| P10 | 总结与行动 | 一页纸战略全景图 |

### 话术模板
```
"我们看到[市场机会]，决定进入[赛道选择]。
我们的致胜策略是[差异化打法]，需要构建[核心能力]。
预期投入[资源需求]，回报[财务预测]。"
```

## 汇报通用原则

### 结构原则
- **金字塔结构**：结论先行，论据支撑
- **MECE原则**：不重复、不遗漏
- **So What**：每页有明确结论

### 数据原则
- **有数据**：核心论点必须有数据支撑
- **可视化**：图表>表格>文字
- **对比性**：有参照物（vs去年/vs竞品/vs目标）

### 时间控制
- **3分钟版**：P1+P5+P10（核心主张+策略+行动）
- **10分钟版**：完整10页快速过
- **30分钟版**：每页深入讨论

## 常见问题应对

| 问题类型 | 应对策略 |
|---------|---------|
| "数据怎么来的？" | 提前准备数据来源说明 |
| "竞品怎么做的？" | 准备竞品对比分析 |
| "风险是什么？" | 主动呈现风险+应对方案 |
| "需要多少资源？" | 准备3个档位预算方案 |

---

## 选择汇报类型

```
当前业务状态
├─ 业务下滑/危机 → 战略破局型
├─ 业务稳定/增长 → 战略规划型
└─ 既有危机又有机会 → 先破局后规划（两阶段）
```

---
**核心理念**：汇报不是展示工作量，而是推动决策
