# 战略解码阶梯（精华版）

---

## 核心框架

**Strategy Cascade**：5步战略级联，从愿景到执行

```
第1步：愿景与目标（Winning Aspiration）
  ↓ "我们要成为谁？"
第2步：赛道选择（Where to Play）
  ↓ "在哪里竞争？"
第3步：致胜策略（How to Win）
  ↓ "凭什么赢？"
第4步：核心能力（Capabilities）
  ↓ "需要什么能力？"
第5步：管理举措（Management Systems）
  "如何落地执行？"
```

---

## 第1步：愿景与目标

### 核心问题
- 我们想成为谁？
- 3-5年后的成功画面是什么？

### 好愿景标准
| 要素 | 说明 |
|------|------|
| 激励性 | 让团队兴奋 |
| 清晰度 | 一句话说清 |
| 可衡量 | 有具体指标 |
| 时间性 | 有截止日期 |

### 示例
```
❌ 模糊："成为行业领先"
✅ 清晰："3年内成为AI+商业培训赛道第一品牌，年营收5000万"
```

---

## 第2步：赛道选择（Where to Play）

### 核心问题
- 服务哪些客户？
- 在哪个市场/品类竞争？
- 做什么、不做什么？

### 分析维度
```
赛道选择
├─ 客户细分（谁是目标客户？）
├─ 地理范围（哪些区域？）
├─ 产品品类（哪些产品线？）
└─ 价值链位置（做哪个环节？）
```

### 选择原则
- **市场够大**：TAM/SAM/SOM评估
- **增速够快**：年增长率>15%
- **能建壁垒**：有差异化空间
- **能力匹配**：自身资源能支撑

---

## 第3步：致胜策略（How to Win）

### 核心问题
- 为什么客户选我们？
- 和竞品的差异是什么？

### 三种通用策略
| 策略 | 核心 | 适用 |
|------|------|------|
| 成本领先 | 价格最低 | 规模型企业 |
| 差异化 | 独特价值 | 有核心能力 |
| 聚焦 | 细分市场 | 资源有限 |

### 差异化来源
```
差异化
├─ 产品差异（功能/质量/设计）
├─ 服务差异（体验/速度/深度）
├─ 品牌差异（认知/情感/信任）
└─ 成本差异（效率/规模/技术）
```

---

## 第4步：核心能力（Capabilities）

### 核心问题
- 实现战略需要什么能力？
- 现有能力和目标的差距？

### 能力地图
```
能力分析
├─ 核心能力（必须自建）
├─ 一般能力（可外包）
└─ 缺失能力（需补齐）
```

### Gap分析
| 能力 | 现状 | 目标 | 差距 | 行动 |
|------|------|------|------|------|
| AI技术 | 2分 | 4分 | 2分 | 外部合作 |
| 课程研发 | 4分 | 5分 | 1分 | 内部提升 |

---

## 第5步：管理举措（Management Systems）

### 核心问题
- 如何组织资源？
- 如何衡量进展？
- 如何激励团队？

### 落地要素
```
管理举措
├─ 组织架构（谁负责什么）
├─ 流程机制（怎么协作）
├─ 考核激励（KPI/OKR）
└─ 资源配置（人/钱/时间）
```

---

## 完整案例：AI培训业务

```
【第1步：愿景】
3年成为AI+商业培训第一品牌，年营收5000万

【第2步：赛道】
- 客户：30+职场中高层管理者
- 市场：中国一二线城市
- 产品：AI+商业场景培训
- 不做：纯AI工具教学

【第3步：致胜策略】
差异化策略：
- 不是教工具，是教场景应用
- Workshop+1v1，深度交付
- 商业管理+AI双重背景

【第4步：核心能力】
- 必须自建：场景化课程设计、高端交付
- 可合作：AI技术深度支持
- 待补齐：品牌影响力

【第5步：管理举措】
- CEO聚焦产品和销售
- 建立会员服务标准
- 月度复盘、季度战略检视
```

---

## 战略一致性检验

```
□ 愿景是否清晰可衡量？
□ 赛道选择是否有数据支撑？
□ 致胜策略是否有差异化？
□ 能力是否匹配战略需求？
□ 管理举措是否能落地？
□ 五步之间是否逻辑一致？
```

## AI实战提示词

```markdown
请你扮演：[战略顾问]，用战略解码阶梯帮我梳理战略。

【我的背景】：[输入背景]
【当前挑战】：[输入挑战]

【你需要做的】：
1. 帮我完成5步战略解码
2. 每步给出关键问题和选择建议
3. 检验五步之间的一致性
4. 输出一页纸战略全景图
```
